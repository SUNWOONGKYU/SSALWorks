# 6편 | CSS - 웹사이트 스타일링

---

HTML 구조를 살펴본 후, 다음 단계로 CSS 스타일링을 시작해봅니다. HTML로 만든 뼈대에 아름다운 디자인을 입히는 CSS의 세계를 살펴봅니다. 스타일시트의 기본 문법부터 박스 모델, 선택자 시스템까지 CSS의 핵심 개념을 이해하고, 반응형 웹사이트 제작까지 실무에 필요한 스타일링 기법을 차근차근 살펴봅니다.

## 1. CSS란 무엇인가

### 1-1 웹사이트를 꾸미는 언어

CSS는 HTML로 만든 웹페이지에 디자인을 입히는 스타일 언어입니다. CSS는 Cascading Style Sheets의 줄임말입니다. HTML이 웹사이트의 뼈대라면 CSS는 그 위에 입히는 옷과 화장이라고 할 수 있습니다.

CSS 없이 HTML만 있는 웹페이지는 흑백 신문처럼 밋밋합니다. 모든 글씨가 검은색에 같은 크기와 같은 폰트로 표시됩니다. CSS가 있어야 다양한 스타일을 조절할 수 있습니다. 색상과 크기를 조절하고 폰트와 배치도 설정할 수 있습니다. 간격도 조절해서 보기 좋고 사용하기 편한 웹사이트가 됩니다.

### 1-2 스타일시트의 개념

CSS의 정식 이름에서 Style Sheets는 스타일 규칙서라는 뜻입니다. 마치 회사의 복장 규정과 같습니다. 웹페이지의 각 요소가 어떤 모양으로 보여져야 하는지 규칙을 정해놓은 것입니다.

예를 들어 모든 제목은 파란색으로 표시할 수 있습니다. 본문은 회색으로 표시하고 링크는 빨간색으로 표시할 수도 있습니다. 이런 규칙을 CSS로 만들어두면 웹브라우저가 이 규칙을 따라서 화면에 표시해줍니다.

### 1-3 HTML과 CSS의 협력 관계

HTML은 요소의 의미를 표시합니다. 이것이 제목인지 본문인지 알려줍니다. CSS는 모양을 지정합니다. 제목을 크고 굵게 만들거나 본문을 작고 연하게 만듭니다. 이 둘이 함께 작동해서 의미와 디자인을 모두 갖춘 완성된 웹페이지가 만들어집니다.

중요한 원칙은 HTML과 CSS의 역할을 분리하는 것입니다. HTML은 구조와 의미만 담당합니다. 모든 디자인은 CSS에서 처리해야 합니다.

## 2. CSS 기본 구조

### 2-1 CSS 규칙의 구성

CSS는 선택자와 선언 블록으로 구성됩니다. 선택자는 스타일을 적용할 HTML 요소를 지정합니다. 선언 블록에는 구체적인 스타일 규칙들이 들어갑니다.

기본 형태는 선택자와 속성 그리고 값으로 구성됩니다. `선택자 { 속성: 값; }` 형태로 작성합니다. 예를 들어 `h1 { color: blue; }`라고 쓰면 모든 h1 태그를 파란색으로 표시하라는 의미입니다.

### 2-2 선택자의 역할

선택자는 CSS 스타일을 어떤 HTML 요소에 적용할지 결정합니다. 다양한 방식으로 요소를 선택할 수 있습니다. 태그 이름으로 선택할 수도 있고 클래스나 ID로도 선택할 수 있습니다.

가장 기본적인 선택자는 태그 선택자입니다. `p`라고 쓰면 모든 p 태그에 스타일이 적용됩니다. `h2`라고 쓰면 모든 h2 태그에 스타일이 적용됩니다.

### 2-3 속성과 값

선언 블록 안에는 속성과 값 쌍이 들어갑니다. color는 글자 색상을 설정하는 속성입니다. font-size는 글자 크기를 설정합니다. background는 배경을 설정하는 속성입니다.

각 속성마다 사용할 수 있는 값이 정해져 있습니다. color에는 색상 이름을 사용할 수 있습니다. 색상 코드로도 설정할 수 있습니다. font-size에는 픽셀이나 퍼센트 등의 크기 단위를 사용합니다.

## 3. CSS를 HTML에 연결하는 방법

### 3-1 외부 스타일시트 방식

가장 권장되는 방식은 CSS를 별도 파일로 만드는 것입니다. 그 파일을 HTML에 연결합니다. HTML의 head 부분에 `<link>` 태그를 사용해서 CSS 파일을 연결합니다.

이 방식이 **정말 큰 장점**을 가지고 있습니다. 여러 HTML 파일이 하나의 CSS 파일을 **함께 공유**할 수 있다는 것입니다. 마치 한 벌의 단체복을 여러 명이 동시에 입는 것처럼 말입니다. 그러면 **웹사이트 전체**의 디자인을 한 번에 관리할 수 있고, 이렇게 하면 **일관성 있고 전문적인** 디자인을 만들기가 훨씬 쉽다는 것입니다.

### 3-2 내부 스타일시트 방식

HTML 파일의 head 부분에 `<style>` 태그를 사용하는 방법도 있습니다. 이 방식은 CSS를 직접 작성하는 방법입니다. 그 HTML 파일에서만 사용되는 특별한 스타일이 있을 때 사용합니다.

### 3-3 인라인 스타일 방식

HTML 태그에 style 속성을 직접 추가하는 방법도 있습니다. 특정 요소 하나만 다르게 스타일링하고 싶을 때 사용합니다. 하지만 가능한 한 피하는 것이 좋습니다.

## 4. 주요 CSS 속성들

### 4-1 텍스트 관련 속성

글자와 관련된 스타일을 설정하는 속성들입니다.

color 속성은 글자 색상을 설정합니다. 색상 이름을 사용할 수 있습니다. red나 blue 같은 이름입니다. 16진수 색상 코드도 사용할 수 있습니다. #ff0000 같은 코드입니다. RGB 값도 사용 가능합니다.

font-size는 글자 크기를 설정합니다. font-family는 글꼴을 설정합니다. font-weight는 글자 굵기를 결정합니다. font-style은 기울임체 여부를 결정합니다.

text-align으로 텍스트 정렬을 조절할 수 있습니다. line-height로 줄 간격도 조절 가능합니다.

### 4-2 배경 관련 속성

요소의 배경을 꾸미는 속성들입니다.

background-color는 배경 색상을 설정합니다. background-image는 배경 이미지를 설정합니다. background-repeat로 이미지 반복 여부를 조절할 수 있습니다. background-position으로 이미지 위치도 조절할 수 있습니다.

### 4-3 크기와 간격 속성

width로 요소의 너비를 설정합니다. height로 높이를 설정합니다. 다양한 단위를 사용할 수 있습니다. 픽셀은 px로 표기합니다. 퍼센트는 %로 표기합니다. em 단위도 사용 가능합니다.

margin은 요소 바깥쪽 간격을, padding은 요소 안쪽 간격을 설정합니다. border는 테두리를 만들 때 사용합니다.

```
[CSS 박스 모델]

        margin (바깥 여백)
    ┌─────────────────────────┐
    │                         │
    │   border (테두리)       │
    │ ┌─────────────────────┐ │
    │ │                     │ │
    │ │  padding (안쪽 여백) │ │
    │ │ ┌─────────────────┐ │ │
    │ │ │                 │ │ │
    │ │ │    Content      │ │ │
    │ │ │    (내용)       │ │ │
    │ │ │                 │ │ │
    │ │ └─────────────────┘ │ │
    │ │                     │ │
    │ └─────────────────────┘ │
    │                         │
    └─────────────────────────┘
```

### 4-4 레이아웃 속성

요소들을 화면에 어떻게 배치할지 결정하는 속성들입니다.

display 속성은 요소가 어떤 방식으로 표시될지 결정합니다. block은 한 줄 전체를 차지하고, inline은 필요한 공간만 차지합니다.

position 속성으로 요소의 위치 지정 방식을 설정하고, float 속성으로 요소를 좌우로 정렬할 수 있습니다.

## 5. CSS 선택자 심화

### 5-1 기본 선택자들

태그 선택자는 특정 HTML 태그를 선택합니다. `h1`이라고 쓰면 모든 h1 태그가 선택됩니다.

클래스 선택자는 점(.)을 사용해서 특정 클래스를 가진 요소들을 선택합니다. `.highlight`라고 쓰면 class="highlight"인 모든 요소가 선택됩니다.

ID 선택자는 샵(#)을 사용해서 특정 ID를 가진 요소를 선택합니다. `#header`라고 쓰면 id="header"인 요소가 선택됩니다.

### 5-2 결합 선택자

여러 선택자를 조합해서 더 정확하게 요소를 선택할 수 있습니다.

하위 선택자는 공백으로 구분합니다. `div p`라고 쓰면 div 안에 있는 모든 p 태그가 선택됩니다.

자식 선택자는 `>`를 사용합니다. `div > p`는 div의 직접적인 자식인 p 태그만 선택합니다.

### 5-3 가상 선택자

특별한 상태나 위치에 있는 요소를 선택하는 선택자들입니다.

`:hover`는 마우스를 올렸을 때, `:active`는 클릭했을 때, `:focus`는 포커스되었을 때의 상태를 선택합니다.

`:first-child`는 첫 번째 자식 요소를, `:last-child`는 마지막 자식 요소를 선택합니다.

## 6. 박스 모델 이해하기

### 6-1 박스 모델의 개념

CSS에서 모든 HTML 요소는 직사각형 박스로 취급됩니다. 이 박스는 내용(content), 패딩(padding), 테두리(border), 마진(margin)으로 구성됩니다.

내용 영역에는 텍스트나 이미지 같은 실제 콘텐츠가 들어갑니다. 패딩은 내용과 테두리 사이의 안쪽 여백이고, 마진은 테두리 바깥쪽 여백입니다.

### 6-2 박스 크기 계산

요소의 실제 크기는 width + padding + border + margin을 모두 합한 것입니다. width: 200px에 padding: 10px, border: 2px를 설정하면 실제로는 224px의 공간을 차지합니다.

이런 계산을 단순화하려면 box-sizing 속성을 border-box로 설정할 수 있습니다. 그러면 padding과 border가 width에 포함되어서 계산이 쉬워집니다.

### 6-3 마진과 패딩 활용

margin과 padding은 top, right, bottom, left 각각 다르게 설정할 수 있습니다. `margin: 10px 20px`이라고 쓰면 위아래는 10px, 좌우는 20px 마진이 적용됩니다.

마진은 인접한 요소들 사이의 거리를 조절할 때, 패딩은 요소 내부의 여백을 만들 때 사용합니다.

## 7. CSS로 반응형 웹사이트 만들기

### 7-1 다양한 화면 크기에 대응하기

요즘에는 스마트폰, 태블릿, PC 등 다양한 기기로 웹사이트를 이용합니다. 기기마다 화면 크기가 다르기 때문에, 모든 기기에서 보기 좋은 디자인을 만들어야 합니다.

반응형 디자인은 화면 크기에 따라 레이아웃이 자동으로 조절되는 디자인 방식입니다. PC에서는 3단 구성으로 보이던 내용이 스마트폰에서는 1단으로 세로 배열되는 식입니다.

### 7-2 미디어 쿼리로 스타일 분기하기

미디어 쿼리는 화면 크기나 기기 종류에 따라 다른 CSS 스타일을 적용하는 기능입니다.

`@media screen and (max-width: 768px)`라고 쓰면 화면 너비가 768px 이하일 때만 적용되는 스타일을 만들 수 있습니다. 주로 태블릿이나 모바일용 스타일을 만들 때 사용합니다.

### 7-3 유연한 크기 단위 사용하기

반응형 디자인에서는 고정된 픽셀 크기보다 퍼센트나 em 같은 상대적 단위를 많이 사용합니다. 이렇게 하면 화면 크기가 바뀌어도 비율에 맞춰서 크기가 조절됩니다.

max-width 속성을 사용하면 최대 크기를 제한할 수 있어서 큰 화면에서 너무 넓어지는 것을 방지할 수 있습니다.
