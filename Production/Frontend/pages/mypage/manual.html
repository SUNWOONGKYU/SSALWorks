1. 시작하기

1-1. 회원가입하기

  1. SSAL Works 웹사이트에 접속합니다.
  2. “회원가입” 버튼을 클릭합니다.
  3. 필수 정보를 입력합니다:
    • 이메일: 고유한 이메일 주소
    • 비밀번호: 8자 이상 (영문+숫자+특수문자)
    • 닉네임: 2-20자
    • 실명: 2-50자 (필수 - 개설비 입금자명 확인, 환불, 세금계산서 발행에 사용)
  4. 약관에 동의합니다:
    • ✅ 서비스 이용약관 (필수)
    • ✅ 개인정보 처리방침 (필수)
    • ☑️ 마케팅 정보 수신 (선택)
  5. “회원가입” 버튼을 클릭합니다.
  6. 회원가입 완료 후 회원 ID(8자리 영숫자)가 발급됩니다.

1-2. 로그인하기

  1. SSAL Works 웹사이트에 접속합니다.
  2. 우측 상단의 “로그인” 버튼을 클릭합니다.
  3. Google 계정으로 로그인합니다.
  4. 빌더 계정이 활성화되어 있으면 대시보드가 표시됩니다.

1-3. 대시보드 화면 구성

대시보드는 5개 영역으로 구성되어 있습니다:

┌────────────────┬─────────────────────────┬────────────────┐
│                │                         │                │
│   좌측 사이드바  │   중앙 상단              │   우측 사이드바  │
│   (프로젝트)    │   (Control Space)       │   (학습/질문)   │
│                ├─────────────────────────┤                │
│                │   중앙 하단              │                │
│                │   (SAL Grid)            │                │
│                │                         │                │
└────────────────┴─────────────────────────┴────────────────┘
  • 좌측 사이드바: 프로젝트 목록, 진행 프로세스
  • 중앙 상단: Order Sheet 작성 공간
  • 중앙 하단: PROJECT SAL GRID
  • 우측 사이드바: Books, Tips, AI Q&A, Sunny에게 묻기

2. 프로젝트 관리

2-1. 새 프로젝트 등록하기

  1. 좌측 사이드바에서 “+ 새로운 Project 등록” 버튼을 클릭합니다.
  2. 프로젝트명을 입력합니다. (예: “쇼핑몰 웹사이트”)
  3. 프로젝트 설명을 입력합니다. (선택사항)
  4. “등록” 버튼을 클릭합니다.
  5. 프로젝트가 생성되고 SAL Grid가 자동으로 초기화됩니다.

2-2. 프로젝트 선택하기

  1. 좌측 사이드바의 “진행중 Project” 섹션에서 작업할 프로젝트를 클릭합니다.
  2. 해당 프로젝트의 SAL Grid와 Order Sheet가 로드됩니다.
  3. 프로젝트명이 상단에 표시됩니다.

2-3. 진행 프로세스 확인하기

  1. 좌측 사이드바의 “진행 프로세스” 섹션을 확인합니다.
  2. 프로세스는 3가지 유형으로 구분됩니다:

☆ 특별 단계 (자동화) - 디렉토리 구조 생성 - PROJECT SAL Grid 생성

○ 준비 단계 (병렬 진행 가능) - 사업계획: 시장조사, 경쟁분석, 사업계획서 - 프로젝트 기획: Project Plan, Requirements, Architecture, UI/UX, DB Design, API Design

S1~S5 실행 단계 (순차 진행 필수) | 단계 | 명칭 | 주요 내용 | |——|——|———-| | S1 | 개발 준비 | Vercel 설정, 환경변수, 도메인 연결, 에러 트래킹 | | S2 | 개발 1차 | OAuth, 이메일, 핵심 API, 회원가입, 프로젝트 관리 | | S3 | 개발 2차 | AI 연동, 구독 권한, AI Q&A UI | | S4 | 개발 3차 | 결제, 관리자 대시보드, 테스트, 크레딧 | | S5 | 개발 마무리 | 프로덕션 배포, QA, 안정화 |

  1. 각 단계를 클릭하면:
    • 하위 작업 목록이 펼쳐집니다
    • Workspace에 Order Sheet 템플릿이 자동 로드됩니다
    • 상태 아이콘으로 진행 상황을 확인할 수 있습니다:
      • ⚪ 대기
      • 🔵 진행 중
      • ✅ 완료

2-4. 안내문 열기

  1. 진행 프로세스에서 단계 옆의 📖 아이콘을 클릭합니다.
  2. 해당 단계의 안내문이 팝업으로 표시됩니다.
  3. 안내문을 참고하여 해당 단계의 작업을 진행합니다.

3. Order Sheet 사용법

3-1. Order Sheet 작성하기

  1. 중앙 상단의 Control Space 영역을 확인합니다.
  2. “Order Sheet 템플릿” 드롭다운에서 템플릿을 선택합니다.
  3. 템플릿이 에디터에 로드됩니다.
  4. 템플릿의 각 항목을 채웁니다:
    • 작업 유형: 어떤 종류의 작업인지
    • 작업 내용: 구체적으로 무엇을 해야 하는지
    • 참고 자료: 참고할 파일이나 문서
    • 기대 결과물: 어떤 결과물이 나와야 하는지

3-2. Order Sheet 복사하여 Claude Code에 전달하기

  1. Order Sheet 작성이 완료되면 “Order Sheet 복사하기” 버튼을 클릭합니다.
  2. Order Sheet 전체 내용이 클립보드에 복사됩니다.
  3. Claude Code 터미널을 엽니다.
  4. 터미널에 Ctrl+V (또는 마우스 우클릭 → 붙여넣기)로 Order Sheet를 붙여넣습니다.
  5. Enter를 눌러 AI에게 작업을 시작하도록 합니다.

3-3. 결과 Report 확인하기

  1. AI가 작업을 완료하면 Reports 폴더에 결과가 저장됩니다.
  2. “Reports 불러오기” 버튼을 클릭합니다.
  3. 파일 목록에서 확인할 Report를 선택합니다.
  4. Report 내용이 화면에 표시됩니다.

4. PROJECT SAL GRID 사용법

4-1. Grid 보기 모드 전환하기

  1. 중앙 하단의 SAL Grid 영역을 확인합니다.
  2. 우측 상단의 보기 모드 버튼을 클릭합니다:
    • 2D Card View: 카드 형태로 Task 표시
    • 3D Block View: 3차원 블록으로 시각화
  3. 풀스크린 버튼을 클릭하면 Grid를 전체 화면으로 볼 수 있습니다.

4-2. Task 상태 확인하기

  1. Grid 상단의 상태별 통계를 확인합니다:
    • Pending: 대기 중인 Task 수
    • In Progress: 진행 중인 Task 수
    • Executed: 실행 완료된 Task 수
    • Completed: 검증까지 완료된 Task 수
    • Fixing: 수정이 필요한 Task 수
  2. 색상으로 상태를 구분합니다:
    • 회색: Pending
    • 파란색: In Progress
    • 주황색: Executed
    • 녹색: Completed
    • 빨간색: Fixing

4-3. Task 상세 정보 보기

  1. Grid에서 Task 카드를 클릭합니다.
  2. Task 상세 정보 패널이 표시됩니다:
    • Task ID: S2F1 형식의 고유 ID
    • Task Name: Task 이름
    • Task Instruction: 작업 지침
    • Task Agent: 담당 AI 에이전트
    • Generated Files: 생성된 파일 목록
    • Verification Status: 검증 상태

4-4. Stage Gate 검증하기

  1. 현재 Stage의 모든 Task가 Completed 상태인지 확인합니다.
  2. “Stage Gate 검증” 버튼을 클릭합니다.
  3. AI가 자동으로 검증을 수행합니다.
  4. 검증 결과를 확인합니다.
  5. 문제가 없으면 “승인” 버튼을 클릭하여 다음 Stage로 진행합니다.

5. 학습용 Books 사용법

5-1. Books 열람하기

  1. 우측 사이드바에서 “📚 학습용 Books” 섹션을 클릭합니다.
  2. 3개의 Book 중 원하는 것을 선택합니다:
    • 제1권: Claude/Claude Code 사용법 (34편)
    • 제2권: 풀스택 웹사이트 개발 기초지식 (34편)
    • 제3권: 프로젝트 관리 방법 (12편)
  3. 편(Chapter) 목록에서 읽고 싶은 편을 클릭합니다.
  4. 내용이 팝업으로 표시됩니다.

5-2. Books 검색하기

  1. “📚 학습용 Books” 섹션의 검색창을 클릭합니다.
  2. 찾고 싶은 키워드를 입력합니다. (예: “API”, “Supabase”)
  3. 검색 결과가 표시됩니다.
  4. 원하는 결과를 클릭하여 내용을 확인합니다.

6. 외부 연동 설정

6-1. 설정 가이드 열기

  1. 우측 사이드바에서 “🔧 외부 연동 설정” 섹션을 클릭합니다.
  2. 5가지 설정 가이드 중 원하는 것을 선택합니다:
    • 🗄️ 데이터베이스 설정 (Supabase)
    • 🔐 회원인증 설정 (Google OAuth)
    • 📧 이메일 시스템 설정 (Resend)
    • 🚀 배포/도메인 설정 (Vercel)
    • 💳 결제 시스템 설정 (토스페이먼츠)

6-2. 설정 따라하기

  1. 가이드의 단계별 지침을 읽습니다.
  2. 각 단계의 체크박스를 완료할 때마다 체크합니다.
  3. 필요한 키(API Key, Secret 등)를 기록해둡니다.
  4. 모든 단계를 완료하면 연동이 활성화됩니다.

7. Tips 활용법

7-1. Tips 열람하기

  1. 우측 사이드바에서 “💡 Tips” 섹션을 클릭합니다.
  2. 16개 카테고리 중 원하는 것을 선택합니다.
  3. 카테고리 내 Tips 목록이 표시됩니다.
  4. 원하는 Tip을 클릭하여 내용을 확인합니다.

7-2. Tips 검색하기

  1. “💡 Tips” 섹션의 검색창을 클릭합니다.
  2. 찾고 싶은 키워드를 입력합니다. (예: “에러”, “Git”)
  3. 검색 결과에서 관련 Tip을 클릭합니다.

8. 다른 AI에게 묻기

8-1. AI에게 질문하기

  1. 우측 사이드바에서 “🤖 다른 AI에게 묻기” 섹션을 클릭합니다.
  2. AI 서비스를 선택합니다:
    • ChatGPT: 일반적인 개발 질문
    • Gemini: 코드 리뷰, 설계 질문
    • Perplexity: 최신 기술 정보 검색
  3. 질문 입력창에 질문을 작성합니다.
  4. “질문하기” 버튼을 클릭합니다.
  5. AI의 답변이 표시됩니다.

8-2. 크레딧 확인하기

  1. 대시보드 상단의 크레딧 잔액을 확인합니다.
  2. AI 질문 시 크레딧이 차감됩니다.
  3. 잔액이 부족하면 “크레딧 충전”을 클릭하여 충전합니다.

8-3. 질문/답변 이력 보기

  1. “질문/답변 이력” 버튼을 클릭합니다.
  2. 과거 질문과 AI 답변 목록이 표시됩니다.
  3. 원하는 항목을 클릭하여 상세 내용을 확인합니다.

9. Sunny에게 묻기

9-1. 플랫폼 사용 문의하기

  1. 우측 사이드바에서 “👤 Sunny에게 묻기” 섹션을 클릭합니다.
  2. 문의 유형을 선택합니다:
    • 사용법 문의: 플랫폼 사용 방법
    • 문제 해결: 개발 중 발생한 문제
    • 버그 신고: 발견한 버그
    • 기능 제안: 새로운 기능 아이디어
  3. 질문 내용을 작성합니다.
  4. 필요시 파일을 첨부합니다.
  5. “문의하기” 버튼을 클릭합니다.

9-2. 비즈니스 멘토링 받기

  1. 문의 유형에서 “비즈니스 멘토링”을 선택합니다.
  2. 멘토링 주제를 선택합니다:
    • 사업계획서 검토
    • 사업 전략 자문
    • 비즈니스 모델 검토
  3. 상세 내용과 질문을 작성합니다.
  4. 관련 자료가 있으면 첨부합니다.
  5. “문의하기” 버튼을 클릭합니다.

9-3. 회계/세무 자문 받기

  1. 문의 유형에서 “회계/세무 자문”을 선택합니다.
  2. 자문 주제를 선택합니다:
    • 세무 관련 질문
    • 회계 처리 안내
  3. 질문 내용을 구체적으로 작성합니다.
  4. “문의하기” 버튼을 클릭합니다.

10. 마이페이지 관리

10-1. 프로필 수정하기

  1. 좌측 사이드바에서 “마이페이지”를 클릭합니다.
  2. “내 프로필” 탭을 선택합니다.
  3. 이름, 연락처 등을 수정합니다.
  4. “저장” 버튼을 클릭합니다.

10-2. 플랫폼 이용 현황 확인 및 관리하기

  1. “서비스 이용 현황” 탭을 선택합니다.
  2. 현재 플랫폼 이용 상태를 확인합니다:
    • 이용 시작일
    • 다음 결제일
    • 월 이용료: ₩50,000/월 (VAT 포함, 첫 3개월 무료)

결제 수단 등록: 1. 개설비 납부 후 결제 수단을 등록합니다. 2. 선택 가능한 결제 수단: - 계좌 자동이체: 은행 선택, 계좌번호, 예금주 - 카드 자동결제: 카드번호, 유효기간, CVC

자동결제: - 첫 3개월 무료 기간 이후, 매월 개설비 납부일 기준으로 동일 날짜에 자동결제됩니다. - 결제 3일 전 알림이 발송됩니다. - 결제 실패 시 2회 재시도 후 서비스가 일시정지됩니다.

이용 일시정지: 1. “이용 일시정지” 버튼을 클릭합니다. 2. 일시정지 기간 동안 이용료가 청구되지 않습니다. 3. 제한 사항: - ❌ 새 프로젝트 등록 불가 - ❌ Order Sheet 작성 불가 - ✅ 기존 프로젝트 조회 가능 - ✅ Books, FAQ 이용 가능

이용 해지: 1. “이용 해지” 버튼을 클릭합니다. 2. 해지 시: - 개설비는 환불되지 않습니다. - 이번 달 이용료는 일할 계산 환불됩니다. - ⚠️ 재가입 시 개설비를 다시 납부해야 합니다.

10-3. 크레딧 관리하기

  1. “크레딧 관리” 탭을 선택합니다.
  2. 현재 잔액을 확인합니다.
  3. “사용 내역”을 클릭하여 상세 내역을 확인합니다.

충전 금액: - ₩10,000 - ₩30,000 - ₩50,000 - ₩100,000

충전 방법: 1. “충전하기” 버튼을 클릭합니다. 2. 충전 금액을 선택합니다. 3. 토스페이먼츠 결제창에서 결제를 진행합니다. 4. 결제 완료 후 즉시 크레딧이 충전됩니다.
