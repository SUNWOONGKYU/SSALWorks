1. 시작하기

1-1. 회원가입하기

  1. SSAL Works 웹사이트에 접속합니다.
  2. "회원가입" 버튼을 클릭합니다.
  3. 필수 정보를 입력합니다:
    • 이메일: 고유한 이메일 주소
    • 비밀번호: 8자 이상 (영문+숫자+특수문자)
    • 닉네임: 2-20자
    • 실명: 2-50자 (필수 - 개설비 입금자명 확인, 환불, 세금계산서 발행에 사용)
  4. 약관에 동의합니다:
    • ✅ 서비스 이용약관 (필수)
    • ✅ 개인정보 처리방침 (필수)
    • ☑️ 마케팅 정보 수신 (선택)
  5. "회원가입" 버튼을 클릭합니다.
  6. 회원가입 완료 후 회원 ID(8자리 영숫자)가 발급됩니다.

1-2. 로그인하기

  1. SSAL Works 웹사이트에 접속합니다.
  2. 우측 상단의 "로그인" 버튼을 클릭합니다.
  3. Google 계정으로 로그인합니다.
  4. 빌더 계정이 활성화되어 있으면 대시보드가 표시됩니다.

1-3. 대시보드 화면 구성

대시보드는 5개 영역으로 구성되어 있습니다:

┌────────────────┬─────────────────────────┬────────────────┐
│                │                         │                │
│   좌측 사이드바  │   중앙 상단              │   우측 사이드바  │
│   (프로젝트)    │   (Control Space)       │   (학습/질문)   │
│                ├─────────────────────────┤                │
│                │   중앙 하단              │                │
│                │   (SAL Grid)            │                │
│                │                         │                │
└────────────────┴─────────────────────────┴────────────────┘
  • 좌측 사이드바: 프로젝트 목록, 진행 프로세스
  • 중앙 상단: Order Sheet 작성 공간
  • 중앙 하단: PROJECT SAL GRID
  • 우측 사이드바: Books, Tips, AI Q&A, Sunny에게 묻기

2. 프로젝트 관리

2-1. 새 프로젝트 등록하기

  1. 좌측 사이드바에서 "+ 새로운 Project 등록" 버튼을 클릭합니다.
  2. 프로젝트명을 입력합니다. (예: "쇼핑몰 웹사이트")
  3. 프로젝트 설명을 입력합니다. (선택사항)
  4. "등록" 버튼을 클릭합니다.
  5. 프로젝트가 생성되고 SAL Grid가 자동으로 초기화됩니다.

2-2. 프로젝트 선택하기

  1. 좌측 사이드바의 "진행중 Project" 섹션에서 작업할 프로젝트를 클릭합니다.
  2. 해당 프로젝트의 SAL Grid와 Order Sheet가 로드됩니다.
  3. 프로젝트명이 상단에 표시됩니다.

2-3. 진행 프로세스 확인하기

S1~S5 실행 단계 (순차 진행 필수)

단계명칭주요 내용
S1개발 준비Vercel 설정, 환경변수, 도메인 연결
S2개발 1차OAuth, 이메일, 핵심 API, 회원가입
S3개발 2차AI 연동, 구독 권한, AI Q&A UI
S4개발 3차결제, 관리자 대시보드, 테스트
S5개발 마무리프로덕션 배포, QA, 안정화

3. Order Sheet 사용법

3-1. Order Sheet 작성하기

  1. 중앙 상단의 Control Space 영역을 확인합니다.
  2. "Order Sheet 템플릿" 드롭다운에서 템플릿을 선택합니다.
  3. 템플릿이 에디터에 로드됩니다.
  4. 템플릿의 각 항목을 채웁니다:
    • 작업 유형: 어떤 종류의 작업인지
    • 작업 내용: 구체적으로 무엇을 해야 하는지
    • 참고 자료: 참고할 파일이나 문서
    • 기대 결과물: 어떤 결과물이 나와야 하는지

3-2. Claude Code에 전달하기

  1. Order Sheet 작성 완료 후 "Order Sheet 복사하기" 버튼을 클릭합니다.
  2. Claude Code 터미널에 Ctrl+V로 붙여넣습니다.
  3. Enter를 눌러 AI에게 작업을 시작하도록 합니다.

4. PROJECT SAL GRID 사용법

4-1. Grid 보기 모드

  • 2D Card View: 카드 형태로 Task 표시
  • 3D Block View: 3차원 블록으로 시각화

4-2. Task 상태 확인하기

색상상태설명
회색Pending대기 중
파란색In Progress진행 중
주황색Executed실행 완료
녹색Completed검증 완료
빨간색Fixing수정 필요

5. 학습용 Books 사용법

5-1. Books 열람하기

  1. 우측 사이드바에서 "📚 학습용 Books" 섹션을 클릭합니다.
  2. 3개의 Book 중 원하는 것을 선택합니다:
    • 제1권: Claude/Claude Code 사용법 (34편)
    • 제2권: 풀스택 웹사이트 개발 기초지식 (34편)
    • 제3권: 프로젝트 관리 방법 (12편)

6. 외부 연동 설정

6-1. 설정 가이드

  • 🗄️ 데이터베이스 설정 (Supabase)
  • 🔐 회원인증 설정 (Google OAuth)
  • 📧 이메일 시스템 설정 (Resend)
  • 🚀 배포/도메인 설정 (Vercel)
  • 💳 결제 시스템 설정 (토스페이먼츠)

10. 마이페이지 관리

10-1. 서비스 이용 현황

10-2. 크레딧 관리

충전 금액: ₩10,000 / ₩30,000 / ₩50,000 / ₩100,000

이용 해지 시 주의: 개설비는 환불되지 않습니다. 재가입 시 개설비를 다시 납부해야 합니다.
