# 구체적으로 질문하는 방법

## 💡 Tip

"코드가 안 돼요"보다 **구체적인 정보**를 포함하면 더 정확한 답변을 받을 수 있습니다.

## 좋은 질문의 요소

1. **무엇을 하려고 했는가** (기대 동작)
2. **실제로 무슨 일이 일어났는가** (실제 동작)
3. **어떤 오류 메시지가 나왔는가** (에러)
4. **어떤 코드/파일에서 발생했는가** (위치)

## 예시 비교

### ❌ 나쁜 질문
```
"로그인이 안 돼요"
```

### ✅ 좋은 질문
```
"Google 로그인 버튼 클릭 시 콘솔에 다음 오류가 발생합니다:

오류: 'Unsupported provider: provider is not enabled'
파일: pages/auth/login.html
브라우저: Chrome 120
시도한 것: 팝업 차단 해제, 시크릿 모드 시도

기대 동작: Google 로그인 화면으로 이동
실제 동작: 오류 메시지 표시 후 아무 일도 안 됨"
```

## 정보 수집 방법

**1. 브라우저 콘솔 열기**
- Windows: `F12` → Console 탭
- Mac: `Cmd + Option + J`

**2. 오류 메시지 복사**
- 빨간색 오류 메시지 전체 복사

**3. 네트워크 탭 확인**
- API 호출 실패 여부
- HTTP 상태 코드

## 코드 공유 시

```javascript
// 파일: pages/auth/login.html
// 라인: 612

const { data, error } = await supabaseClient.auth.signInWithOAuth({
    provider: 'google',
    options: {
        redirectTo: window.location.origin + '/index.html'  // 여기서 문제 발생
    }
});
```

- Google Cloud Console 설정
