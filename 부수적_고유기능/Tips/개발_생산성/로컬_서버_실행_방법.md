# 로컬 서버 실행 방법

## 질문
로컬에서 웹사이트를 테스트하려면 어떻게 해야 하나요?

## 답변

### 로컬 서버가 필요한 이유

- `file://` 프로토콜로는 OAuth, API 호출 등이 작동하지 않음
- 반드시 `http://localhost:xxxx`로 테스트 필요

### 간단한 방법: npx serve

```bash
# Production 폴더로 이동
cd Production/Frontend

# serve 실행 (포트 8888)
npx serve . -l 8888

# 브라우저에서 접속
http://localhost:8888
```

### 다른 방법들

**Python 사용**
```bash
# Python 3
python -m http.server 8888

# Python 2
python -m SimpleHTTPServer 8888
```

**VS Code Live Server**
1. VS Code 확장 "Live Server" 설치
2. HTML 파일 우클릭 → "Open with Live Server"

### SSAL Works 테스트 URL

```
메인 페이지: http://localhost:8888/index.html
로그인: http://localhost:8888/pages/auth/login.html
Google 로그인: http://localhost:8888/pages/auth/google-login.html
```

### 포트 충돌 시

```bash
# 다른 포트 사용
npx serve . -l 3000

# 사용 중인 포트 확인 (Windows)
netstat -ano | findstr :8888

# 사용 중인 포트 확인 (Mac/Linux)
lsof -i :8888
```
