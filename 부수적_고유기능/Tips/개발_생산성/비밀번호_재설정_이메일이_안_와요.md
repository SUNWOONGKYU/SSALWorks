# 비밀번호 재설정 이메일이 안 와요

## 질문
비밀번호 찾기를 했는데 이메일이 오지 않아요.

## 답변

### 확인할 사항

**1. 스팸 메일함 확인**
- 발신자: noreply@mail.app.supabase.io
- 제목: "Reset your password"

**2. 이메일 주소 확인**
- 가입 시 사용한 이메일과 동일한지 확인
- 오타가 없는지 확인

**3. Resend API 설정 확인 (관리자용)**

이메일 발송이 전혀 안 되는 경우:
```env
# Vercel 환경변수 확인
RESEND_API_KEY=re_xxx
```

### Supabase 이메일 설정

**기본 설정 (개발용)**
- Supabase 내장 이메일 서비스 사용
- 하루 발송 제한: 4통

**프로덕션 설정 (권장)**
- Resend, SendGrid 등 외부 서비스 연동
- Supabase Dashboard → Project Settings → Auth → SMTP Settings

### 이메일 템플릿 위치

```
Supabase Dashboard → Authentication → Email Templates
- Confirm signup
- Reset password
- Magic link
```

### 재발송 방법

1. 로그인 페이지에서 "비밀번호 찾기" 클릭
2. 이메일 주소 입력
3. "재설정 링크 보내기" 클릭
4. 5분 후 재시도 가능

### 관련 파일
- `Production/Frontend/pages/auth/reset-password.html`
