# 결제 시스템 설정 가이드 (Part A - 초보자용)

> 토스 페이먼트 가입부터 API 키 발급까지 - 단계별 클릭 가이드

**대상**: 비개발자, 처음 설정하는 분
**소요**: 가입 10분 + 가맹점 심사 1-2주

---

## ⚠️ 먼저 알아두세요!

토스 페이먼트 결제 연동은 **가맹점 심사**가 필요합니다.

| 항목 | 내용 |
|------|------|
| 심사 기간 | 약 1-2주 |
| 필요 서류 | 사업자등록증, 대표자 신분증, 통장 사본 |
| 개발 시작 | 심사 전에도 **테스트 키**로 가능 |

---

## 이 가이드에서 할 일

1. ✅ 토스 페이먼트 개발자센터 가입
2. ✅ 테스트 API 키 발급
3. ✅ (선택) 가맹점 심사 신청

**가입과 테스트 키 발급은 클릭만으로 완료됩니다!**

---

## 전체 흐름 이해하기

```
사용자가 구독 결제 버튼 클릭
    ↓
토스 결제 화면에서 카드 정보 입력
    ↓
토스가 카드 검증 후 "빌링키" 발급
    ↓
우리 앱이 빌링키를 저장
    ↓
매월 빌링키로 자동 결제
```

> 💡 **빌링키**: 카드 정보 대신 저장하는 안전한 토큰. 이걸로 정기결제를 합니다.

---

## Phase 1: 토스 페이먼트 가입

### 1-1. 개발자센터 접속
1. 웹 브라우저에서 https://developers.tosspayments.com 접속
2. 우측 상단 **"회원가입"** 클릭

### 1-2. 회원가입
1. 이메일 또는 소셜 로그인으로 가입
2. 이메일 인증 완료
3. 개발자센터 로그인

---

## Phase 2: 테스트 API 키 발급

### 2-1. API 키 확인 페이지로 이동
1. 로그인 후 **"내 개발정보"** 클릭
2. **"API 키"** 탭 클릭

### 2-2. 테스트 키 복사

화면에 다음 2개의 키가 표시됩니다:

| 키 종류 | 형식 | 용도 |
|---------|------|------|
| 클라이언트 키 | test_ck_xxxxxxxxx | 웹사이트 결제 화면 |
| 시크릿 키 | test_sk_xxxxxxxxx | 서버 API 호출 |

두 키 모두 메모장에 복사해두세요.

> ⚠️ **시크릿 키**는 비밀번호처럼 절대 남에게 보여주면 안 돼요!

---

## Phase 3: 가맹점 심사 신청 (선택)

> 💡 개발 단계에서는 테스트 키로 충분합니다.
> 실제 결제를 받으려면 가맹점 심사가 필요합니다.

### 3-1. 심사 신청
1. 개발자센터 → **"가맹점 신청"**
2. 필요 서류 업로드:
   - 사업자등록증
   - 대표자 신분증
   - 통장 사본

### 3-2. 서비스 URL 등록
1. **서비스 URL** 입력 (예: https://www.ssalworks.ai.kr)
2. **콜백 URL** 입력 (결제 완료 후 돌아올 페이지)

### 3-3. 심사 대기
- 심사 기간: 약 1-2주
- 심사 완료 후 **라이브 키** 발급

---

## Phase 4: 환경변수 등록 (Vercel)

### 4-1. Vercel 환경변수 추가

Vercel Dashboard → Settings → Environment Variables

| Key | Value | Environment |
|-----|-------|-------------|
| TOSS_CLIENT_KEY | test_ck_xxx... | Preview, Development |
| TOSS_SECRET_KEY | test_sk_xxx... | Preview, Development |

> 💡 심사 완료 후 Production 환경에 라이브 키 추가

## 자주 묻는 질문

### Q: 테스트 키와 라이브 키의 차이는?

| 키 종류 | 용도 | 실제 결제 |
|---------|------|----------|
| 테스트 키 | 개발/테스트 | ❌ 안 됨 |
| 라이브 키 | 프로덕션 | ✅ 됨 |

### Q: 가맹점 심사 없이 테스트할 수 있나요?

네! 테스트 키로 결제 플로우 전체를 테스트할 수 있습니다.
다만 실제 돈이 빠져나가진 않습니다.

### Q: 심사는 왜 1-2주나 걸리나요?

토스 페이먼트가 다음을 확인합니다:
- 사업자 정보 유효성
- 서비스 내용 적합성
- 불법/사기 가능성 검토
