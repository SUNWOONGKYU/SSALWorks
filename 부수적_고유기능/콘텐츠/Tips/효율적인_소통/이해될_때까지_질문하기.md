# 이해될 때까지 질문하기

## 요약
이해 안 되면 계속 질문. "왜 그렇게 해?", "더 쉽게 설명해줘" 요청. 이해해야 비슷한 상황에서 스스로 해결 가능.

## 상세

### 이해 없이 vs 이해 후

```
X 이해 안 되지만 "알겠어 해줘"
  → 비슷한 상황에서 스스로 해결 못함

O "useCallback이 뭐야? 왜 필요해?"
  → 개념 습득 → 응용 가능
```

### 질문 유형

| 유형 | 예시 |
|------|------|
| 왜 | "왜 이렇게 해야 해?" |
| 어떻게 | "이게 어떻게 동작해?" |
| 뭐야 | "이 용어가 뭐야?" |
| 재설명 | "더 쉽게 설명해줘" |

### 질문 연속 예시

```
1차: "useEffect가 뭐야?"
2차: "의존성 배열이 뭐야?"
3차: "빈 배열이랑 배열 없는 거 뭐가 달라?"
4차: "언제 어떤 걸 써야 해? 예시 보여줘"
```

### 이해 확인

```
"내가 이해한 게 맞는지 확인해줘:
[이해한 내용]
맞아?"

"결론만 세 줄로 요약해줘"
```

### 뭘 물어야 할지 모를 때

```
"솔직히 뭘 모르는지도 모르겠어.
기초부터 다시 설명해줘"
```
