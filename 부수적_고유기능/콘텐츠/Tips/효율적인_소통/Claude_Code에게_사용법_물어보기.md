# Claude Code 기능이 궁금하면 Claude Code 자체에게 직접 물어보기

> 이 문서는 Claude Code의 기능과 사용법을 Claude Code 자체에게 질문하는 방법을 설명합니다.

---

## 왜 Claude Code에게 직접 물어보는가

사용법을 모를 때 문서를 찾는 것보다 직접 물어보는 것이 빠릅니다. Claude Code는 자신의 기능과 사용법을 잘 알고 있습니다.

---

## 질문할 수 있는 것

| 카테고리 | 예시 |
|----------|------|
| 기능 확인 | "파일 생성할 수 있어?" |
| 사용법 | "MCP 서버 어떻게 설정해?" |
| 가능 여부 | "Git 커밋 자동화 가능해?" |
| 비교 | "Skills와 Slash Commands 차이가 뭐야?" |
| 설정 | "CLAUDE.md 파일 역할이 뭐야?" |

---

## 기능 관련 질문 예시

```
"Slash Command 만드는 방법 설명해줘"
"CLAUDE.md 파일 역할이 뭐야?"
"세션 간에 정보 공유하는 방법 있어?"
"MCP 서버란 뭐야? 어떻게 사용해?"
"Skills는 어떻게 활용해?"
```

---

## 문제 해결 질문 예시

```
"파일 저장이 안 돼. 권한 오류가 나는데 어떻게 해결해?"
"MCP 서버 설정했는데 연결 안 돼. 설정 파일 보여줄게"
"터미널 명령어 실행하면 에러 나는데 왜 그래?"
```

---

## 좋은 질문 vs 나쁜 질문

| 나쁜 질문 | 좋은 질문 |
|----------|----------|
| "안 돼" | "파일 저장이 안 돼. 권한 오류가 나는데 어떻게 해결해?" |
| "이상해" | "MCP 서버 설정했는데 연결 안 돼. 설정 파일 보여줄게" |
| "모르겠어" | "Slash Command가 뭔지, 어떻게 만드는지 설명해줘" |

---

## 답변 이해 안 되면

```
"더 쉽게 설명해줘"
"예시 코드 보여줘"
"단계별로 알려줘"
"비개발자도 이해하게 설명해줘"
```
