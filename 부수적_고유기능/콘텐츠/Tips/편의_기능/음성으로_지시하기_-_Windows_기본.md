# Windows 음성 인식(Win+H)으로 타이핑 없이 음성으로 지시하기

> 이 문서는 Windows 기본 음성 입력 기능을 활용하여 Claude Code에 지시하는 방법을 설명합니다.

---

## 사용법

1. Claude Code 터미널 창 선택 (활성화)
2. `Windows + H` 누르기
3. 마이크 아이콘이 활성화되면 말하기
4. 말 멈추면 자동으로 텍스트 변환

---

## 음성 명령어

| 명령어 | 동작 |
|--------|------|
| "마침표" | . 입력 |
| "쉼표" | , 입력 |
| "새 줄" | Enter |
| "삭제" | 마지막 단어 삭제 |
| "스페이스" | 공백 입력 |
| "취소" | 마지막 음성 입력 취소 |

---

## 정확도 향상 팁

- 발음 명확하게 하기
- 평소 대화 속도로 말하기
- 주변 소음 최소화
- 헤드셋 마이크 사용 권장
- 짧은 문장 단위로 끊어 말하기
