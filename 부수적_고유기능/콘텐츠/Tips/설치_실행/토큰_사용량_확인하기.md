# /cost 명령어로 토큰 사용량과 비용 실시간 확인하기

> 이 문서는 Claude Code의 토큰 사용량을 확인하고 효율적으로 관리하는 방법을 설명합니다.

---

## 토큰이란?

Claude가 텍스트를 처리하는 단위입니다. 영어는 약 4글자당 1토큰, 한글은 1-2글자당 1토큰입니다. 한글이 영어보다 토큰을 더 많이 소비합니다.

---

## 사용량 확인

```
/status

📊 사용량 정보
• 현재 사용량: 150,000 / 500,000 토큰
• 리셋까지: 4시간 23분
```

---

## 토큰 계산

| 언어 | 대략적 비율 |
|------|------------|
| 영어 | 4글자 ≈ 1토큰 |
| 한글 | 1-2글자 ≈ 1토큰 |
| 코드 | 가변적 |

---

## 토큰 절약 팁

### 효율적인 요청

```
❌ "이 코드가 뭘 하는지 자세히 설명해줘"
✅ "핵심 기능만 간단히 설명해줘"
```

### 세션 관리

- 긴 대화는 새 세션으로
- 관련 없는 작업은 분리
- 컨텍스트가 쌓이면 정리

### 작업 분산

CLI를 여러 개 띄워서 작업을 분산합니다.
- CLI 1: 프론트엔드
- CLI 2: 백엔드
- CLI 3: 문서화
