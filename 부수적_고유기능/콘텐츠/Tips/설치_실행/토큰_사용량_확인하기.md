# 토큰 사용량 확인하기

## 요약
`/status` 명령어로 현재 사용량과 리셋 시점 확인. 한글은 영어보다 토큰 소비 많음. CLI 여러 개 분산 사용하면 효율 향상.

## 상세

### 사용량 확인

```
/status

📊 사용량 정보
• 현재 사용량: 150,000 / 500,000 토큰
• 리셋까지: 4시간 23분
```

### 토큰 계산

| 언어 | 대략적 비율 |
|------|------------|
| 영어 | 4글자 ≈ 1토큰 |
| 한글 | 1-2글자 ≈ 1토큰 |

### 토큰 절약 팁

- 불필요한 출력 줄이기: "핵심만 간단히"
- 긴 대화는 새 세션으로
- CLI 여러 개로 작업 분산 (프론트엔드/백엔드/문서화)
- 명확하고 구체적인 요청

### 한도 도달 시

1. 리셋 시간까지 대기
2. 새 세션 시작
3. 다른 CLI로 전환
