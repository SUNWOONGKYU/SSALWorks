# 폴더에서 터미널 열어 실행하기

## 핵심 요약

Claude Code를 실행하는 가장 편리한 방법은 해당 작업 디렉터리에서 마우스 우클릭 → "터미널 열기"입니다. Windows Terminal이나 Git Bash가 설치되어 있어야 이 옵션이 나타납니다.

## 가장 편리한 방법: 우클릭 → 터미널 열기 (권장)

### 사용 방법

1. 파일 탐색기에서 프로젝트 폴더 열기
2. 폴더 빈 공간에서 **마우스 우클릭**
3. **"터미널에서 열기"** 클릭
4. `claude` 입력 후 Enter

### 왜 이 방법이 좋은가?

- 클릭 2번으로 바로 터미널 열림
- 자동으로 해당 폴더가 현재 디렉터리
- `cd` 명령어로 이동할 필요 없음
- `.claude/CLAUDE.md` 설정 자동 인식

### 우클릭 메뉴가 없을 때

**Windows Terminal 설치:**
1. Microsoft Store에서 "Windows Terminal" 검색
2. 설치 후 재부팅
3. 우클릭 메뉴에 "터미널에서 열기" 등장

**Git Bash 설치:**
1. https://git-scm.com 에서 Git for Windows 다운로드
2. 설치 시 "Add to context menu" 옵션 체크
3. 우클릭 → "Git Bash Here" 사용 가능

## 다른 실행 방법들

### 방법 2: 주소창에서 실행

1. 파일 탐색기에서 프로젝트 폴더 열기
2. 주소창 클릭 (또는 `Alt + D`)
3. `cmd` 또는 `powershell` 입력 후 Enter
4. `claude` 입력

**장점**: 추가 설치 없이 사용 가능

### 방법 3: VS Code 터미널

1. VS Code로 프로젝트 폴더 열기 (File → Open Folder)
2. `Ctrl + `` (백틱) 으로 터미널 열기
3. `claude` 입력

**장점**: 코드 편집과 Claude Code를 같은 화면에서 사용

### 방법 4: 터미널에서 cd로 이동

```bash
# PowerShell/CMD/Git Bash
cd C:\Projects\my-project
claude
```

**장점**: 어디서든 시작 가능
**단점**: 경로를 직접 입력해야 함

### 방법 5: 바탕화면 바로가기 만들기

1. 바탕화면 우클릭 → 새로 만들기 → 바로 가기
2. 위치 입력:
   ```
   cmd /k "cd /d C:\Projects\my-project && claude"
   ```
3. 이름 지정 후 완료

**장점**: 더블클릭으로 바로 실행
**단점**: 프로젝트마다 바로가기 필요

### 방법 6: Windows Terminal 프로필 설정

1. Windows Terminal 열기
2. `Ctrl + ,` (설정)
3. 새 프로필 추가
4. 시작 디렉터리를 프로젝트 폴더로 설정

```json
{
    "name": "My Project",
    "commandline": "cmd.exe",
    "startingDirectory": "C:\\Projects\\my-project"
}
```

**장점**: 드롭다운에서 프로젝트 선택 가능

## 방법별 비교

| 방법 | 편의성 | 추가 설치 | 추천 상황 |
|------|--------|----------|----------|
| 우클릭 → 터미널 | ⭐⭐⭐⭐⭐ | Windows Terminal | 일반 사용 (권장) |
| 주소창 cmd | ⭐⭐⭐⭐ | 없음 | 터미널 없을 때 |
| VS Code 터미널 | ⭐⭐⭐⭐ | VS Code | 개발 중일 때 |
| cd 명령어 | ⭐⭐⭐ | 없음 | 경로 알 때 |
| 바로가기 | ⭐⭐⭐⭐ | 없음 | 자주 쓰는 프로젝트 |
| Terminal 프로필 | ⭐⭐⭐⭐⭐ | Windows Terminal | 여러 프로젝트 |

## 실행 위치가 중요한 이유

Claude Code는 **현재 디렉터리**를 기준으로 작업합니다.

```bash
# ✅ 올바른 위치
C:\Projects\my-project> claude
# .claude/CLAUDE.md 설정 적용됨
# 프로젝트 파일 접근 가능

# ❌ 잘못된 위치
C:\Users\username> claude
# CLAUDE.md 설정 미적용
# 프로젝트 파일 접근 어려움
```

## 주의사항

- 항상 프로젝트 루트 폴더에서 실행하세요
- `.claude/` 폴더가 있는 위치에서 실행해야 설정이 적용됩니다
- 하위 폴더에서 실행하면 상위의 `.claude/` 설정이 적용되지 않을 수 있음
