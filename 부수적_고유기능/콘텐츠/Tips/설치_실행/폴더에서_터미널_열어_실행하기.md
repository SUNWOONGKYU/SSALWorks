# 폴더에서 터미널 열어 실행하기

> 이 문서는 프로젝트 폴더에서 터미널을 열어 Claude Code를 실행하는 방법을 설명합니다.

---

## 왜 프로젝트 폴더에서 실행해야 하는가

프로젝트 루트에서 실행해야 `.claude/CLAUDE.md` 설정이 적용됩니다. 다른 위치에서 실행하면 프로젝트 규칙이 적용되지 않습니다.

```bash
✅ C:\Projects\my-project> claude
   # .claude/CLAUDE.md 설정 적용됨

❌ C:\Users\username> claude
   # 설정 미적용
```

---

## 가장 편리한 방법 (권장)

1. 파일 탐색기에서 프로젝트 폴더 열기
2. 빈 공간에서 **마우스 우클릭**
3. **"터미널에서 열기"** 클릭
4. `claude` 입력 후 Enter

---

## 다른 방법들

| 방법 | 설명 |
|------|------|
| 주소창에 `cmd` | 파일 탐색기 주소창에서 `cmd` 입력 |
| VS Code 터미널 | `Ctrl + `` 으로 터미널 열기 |
| cd 명령어 | `cd C:\Projects\my-project && claude` |
| PowerShell | 시작 메뉴에서 PowerShell 실행 후 cd |

---

## 우클릭 메뉴가 없을 때

**Windows Terminal 설치 필요**

1. Microsoft Store 열기
2. "Windows Terminal" 검색
3. 설치
4. 재부팅 후 우클릭 메뉴 확인
