# 브라우저 콘솔 에러 확인하기

## 요약
F12 또는 Ctrl+Shift+J로 개발자 도구 열기 → Console 탭에서 빨간색 에러 메시지 복사 → Claude Code에 전달. 정확한 에러 메시지가 문제 해결의 핵심.

## 상세

### 개발자 도구 열기

| 브라우저 | 단축키 |
|----------|--------|
| Chrome/Edge | F12 또는 Ctrl+Shift+J |
| Firefox | F12 또는 Ctrl+Shift+K |
| Mac | Cmd+Option+J |

### 탭 종류와 용도

| 탭 | 용도 | 언제 보나 |
|----|------|----------|
| **Console** | JavaScript 에러, 로그 | 기능 오류 시 |
| **Network** | API 요청/응답 | 데이터 안 나올 때 |
| **Elements** | HTML/CSS 구조 | 화면 레이아웃 문제 |

### Network 탭 사용법

1. F12로 개발자 도구 열기
2. 상단 **Network** 탭 클릭
3. 페이지 새로고침 (F5)
4. 빨간색으로 표시된 요청 클릭
5. **Response** 탭에서 에러 메시지 확인

### 메시지 유형

| 색상 | 의미 |
|------|------|
| 🔴 빨간색 | Error - 심각한 오류 |
| 🟡 노란색 | Warning - 경고 |
| 🔵 파란색 | Info - 정보 |

### 자주 보는 에러

| 에러 | 원인 |
|------|------|
| TypeError | 존재하지 않는 속성 접근 |
| ReferenceError | 정의되지 않은 변수 사용 |
| 404 Not Found | 잘못된 파일 경로 |
| CORS 오류 | 다른 도메인 API 호출 |

### 좋은 에러 보고 예시

```
"로그인 버튼 클릭하니 이런 에러가 나와:

[Console 탭]
Uncaught TypeError: Cannot read property 'user' of null
    at handleLogin (login.js:34:15)

[Network 탭]
POST https://xxx.supabase.co/auth/v1/token
Status: 400 Bad Request
Response: {"error": "invalid_grant"}"
```
