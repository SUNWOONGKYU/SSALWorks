# 인증 코드는 숫자 6자리로

## 요약
이메일 인증, 비밀번호 재설정 등의 인증 코드는 숫자 6자리로 설정한다. 사용자가 입력하기 쉽고 보안도 충분하다.

## 상세

### 왜 숫자 6자리인가?

| 형식 | 입력 편의성 | 보안 |
|------|-------------|------|
| 숫자 4자리 | 매우 쉬움 | 약함 (10,000개) |
| **숫자 6자리** | **쉬움** | **충분 (1,000,000개)** |
| 영문+숫자 | 어려움 | 강함 |

### Claude Code에게 요청

```
"이메일 인증 코드 기능 만들어줘.
- 코드: 숫자 6자리
- 유효시간: 5분
- 시도 횟수: 3회 제한"
```

### 코드 생성 예시

```javascript
// 숫자 6자리 인증 코드 생성
function generateVerificationCode() {
  return Math.floor(100000 + Math.random() * 900000).toString();
}
```

### 보안 강화

- 유효시간: 이메일 5~10분, SMS 3~5분
- 시도 횟수: 3~5회 제한
- 재발송 쿨다운: 60초~120초

### 주의
- 코드는 서버 측에서만 생성 (클라이언트 생성 금지)
- 로그에 인증 코드 기록 금지
- 사용된 코드는 즉시 무효화
