# 최적화 성능 개선 작업 시키기

## 요약
프로젝트가 어느 정도 완성된 시점에 최적화 작업을 요청한다. 코드 정리, 불필요한 파일 제거, 로딩 속도 개선, 번들 크기 축소 등을 Claude Code에게 시킬 수 있다.

## 상세

### 최적화 적합 시점

| 시점 | 설명 |
|------|------|
| 기능 완성 후 | 핵심 기능이 모두 동작할 때 |
| 배포 전 | 프로덕션 환경에 올리기 전 |
| 성능 문제 발견 시 | 느림/버벅거림 발생 시 |

### 최적화 요청 예시

```
"프로젝트 전체 최적화해줘:
- 사용하지 않는 코드 제거
- 성능 개선
- 번들 크기 축소"

"메인 페이지 로딩이 5초 이상 걸려. 3초 이내로 줄여줘"
```

### 영역별 최적화

| 영역 | 요청 예시 |
|------|----------|
| 코드 정리 | "사용하지 않는 함수/변수 제거해줘" |
| 프론트엔드 | "이미지 lazy loading 적용해줘" |
| 백엔드 | "N+1 쿼리 문제 해결해줘" |
| 빌드 | "번들 크기 분석하고 축소해줘" |

### 주의
- 기능 개발이 끝난 후에 최적화 진행
- 최적화 전에 현재 상태 백업/커밋
- 최적화 후 기능 테스트 필수
