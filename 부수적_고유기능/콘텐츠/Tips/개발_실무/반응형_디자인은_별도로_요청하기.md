# 반응형 디자인은 별도로 요청하기

## 요약
반응형 디자인과 모바일 최적화는 자동으로 되지 않는다. "반응형으로 만들어줘", "모바일 최적화해줘"라고 명시적으로 요청해야 한다.

## 상세

### 요청 방법

```
❌ "로그인 페이지 만들어줘"
→ 데스크탑 크기에 맞춰 제작됨, 모바일에서 깨짐

✅ "로그인 페이지 반응형으로 만들어줘"
→ 모바일/태블릿/데스크탑 모두 대응
```

### 상세 요청 예시

```
"상품 목록 페이지를 반응형으로 만들어줘:
- 데스크탑: 4열 그리드
- 태블릿: 2열 그리드
- 모바일: 1열 리스트
- 네비게이션은 모바일에서 햄버거 메뉴로"
```

### 브레이크포인트 지정

```
"다음 브레이크포인트로 반응형 적용해줘:
- 모바일: 480px 이하
- 태블릿: 481px ~ 768px
- 데스크탑: 769px 이상"
```

### 주의
- 반응형은 "기능"이 아니라 "디자인" 작업이므로 별도 요청 필수
- 처음부터 반응형으로 만드는 게 나중에 수정하는 것보다 효율적
