# Claude Code 토큰 사용량 확인하고 비용 관리하기

> 이 문서는 Claude Code 사용 시 토큰 사용량을 확인하고 비용을 효율적으로 관리하는 방법을 설명합니다.

---

## 토큰이란

AI 모델이 텍스트를 처리하는 단위입니다. 대략 영어 1단어 = 1토큰, 한글 1글자 = 2-3토큰입니다.

---

## 비용 발생 구조

| 항목 | 비용 발생 |
|------|----------|
| 입력 (프롬프트) | 토큰당 비용 |
| 출력 (응답) | 토큰당 비용 |
| 파일 읽기 | 읽은 내용만큼 |
| 코드 생성 | 생성된 코드만큼 |

---

## 사용량 확인 방법

Anthropic Console에서 확인:
1. console.anthropic.com 접속
2. Usage 탭 클릭
3. 일별/월별 사용량 확인

---

## 비용 절약 팁

| 방법 | 설명 |
|------|------|
| 구체적 요청 | 불필요한 반복 줄임 |
| 파일 범위 지정 | 전체 읽기 대신 특정 부분 |
| 단계별 진행 | 한 번에 너무 많이 요청 X |
| 세션 정리 | 컨텍스트가 너무 길어지면 새 세션 |

---

## 토큰 많이 쓰는 작업

| 작업 | 토큰 소모 |
|------|----------|
| 대용량 파일 읽기 | 높음 |
| 전체 코드베이스 검색 | 높음 |
| 긴 대화 유지 | 누적됨 |
| 반복 수정 | 높음 |

---

## 토큰 적게 쓰는 방법

```
"src/auth/login.js 파일만 읽어줘"  ← 특정 파일
"전체 코드 읽어줘"  ← 비효율적

"50번째 줄 근처 수정해줘"  ← 구체적
"어딘가에 버그가 있어"  ← 비효율적
```

---

## 예산 설정

Anthropic Console에서 월간 사용 한도 설정 가능:
1. Settings → Billing
2. Usage limits 설정
3. 알림 설정
