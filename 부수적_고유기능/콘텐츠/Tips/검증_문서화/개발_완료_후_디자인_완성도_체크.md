# 개발 완료 후 서비스 개시 전 - 디자인/완성도 체크

## 요약
개발 완료 후 오류 수정 및 보완 작업 중 디자인이 깨지거나 기능이 빠질 수 있다. 수정 작업 묶음마다 디자인 체크 + 완성도 체크 필수.

## 상세

### 문제 상황

```
[개발 완료] → [오류 수정/보완 작업] → [어느새 디자인 깨짐]
                                    → [어느새 기능 누락]
```

### 두 가지 체크

| 체크 | 확인 내용 |
|------|----------|
| 디자인 체크 | 레이아웃 깨짐, 반응형 틀어짐, 스타일 날아감 |
| 완성도 체크 | 기능 누락, 빠진 화면, 미완성 부분 |

### Claude Code에게 요청

```
"수정 작업 후 디자인 깨진 곳 없는지 확인해줘"

"원래 계획이랑 비교해서 빠진 기능 있는지 체크해줘"
```

### 주의
- 수정할 때마다 X → 작업 묶음 단위로 체크
- 서비스 개시 전 최종 체크 필수
