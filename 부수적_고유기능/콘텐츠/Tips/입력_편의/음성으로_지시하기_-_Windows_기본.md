# 음성으로 지시하기 - Windows 기본

## 요약
`Windows + H`로 음성 입력 활성화. 말하면 자동으로 텍스트 변환. 타이핑보다 2-3배 빠름. 기술 용어 인식은 AI 서비스가 더 정확.

## 상세

### 사용법

1. Claude Code 터미널 창 선택
2. `Windows + H` 누르기
3. 마이크 아이콘 활성화되면 말하기
4. 말 멈추면 자동 텍스트 변환

### 음성 명령어

| 명령어 | 동작 |
|--------|------|
| "마침표" | . 입력 |
| "쉼표" | , 입력 |
| "새 줄" | Enter |
| "삭제" | 마지막 단어 삭제 |

### 정확도 향상 팁

- 발음 명확하게
- 평소 대화 속도로
- 주변 소음 최소화
- 헤드셋 마이크 권장

### 장단점

| 장점 | 단점 |
|------|------|
| 추가 설치 불필요 | 기술 용어 인식 오류 |
| 빠른 실행 (Win+H) | 영어 혼용 어려움 |
| 무료 | 주변 소음에 민감 |
