# 음성으로 지시하기 - Windows 기본

## 핵심 요약

`Windows + H` 키를 누르면 Windows 기본 음성 입력이 활성화됩니다. 마이크가 연결되어 있으면 말로 지시한 내용이 자동으로 텍스트로 변환되어 Claude Code에 입력됩니다.

## 왜 음성 입력이 유용한가?

### 1. 타이핑보다 빠른 입력 속도

- 평균 타이핑: 분당 40~60단어
- 평균 말하기: 분당 120~150단어
- **2~3배 빠른 입력 속도**

### 2. 긴 지시사항 전달에 적합

```
❌ 타이핑으로 설명하기 힘든 경우:
"이 함수는 사용자 입력을 받아서 유효성 검사를 하고
이메일 형식이 맞는지 확인하고 비밀번호가 8자 이상인지
확인하고 특수문자가 포함되어 있는지 확인해서
모든 조건이 맞으면 true를 반환하고..."

✅ 음성으로 자연스럽게:
"사용자 회원가입 폼 유효성 검사 함수 만들어줘.
이메일은 형식 검증하고, 비밀번호는 8자 이상에
특수문자 포함 필수야."
```

### 3. 손 피로도 감소

장시간 코딩 작업 시 손목과 손가락의 피로를 줄일 수 있습니다.

## 사용 방법

### 기본 사용법

1. Claude Code가 실행된 터미널 창 선택
2. **`Windows + H`** 키 누르기
3. 화면 상단에 음성 입력 도구 모음 표시
4. 마이크 아이콘이 활성화되면 말하기 시작
5. 말을 멈추면 자동으로 텍스트 변환

### 음성 입력 도구 모음

```
┌────────────────────────────────────┐
│  🎤  [일시 중지]  [설정]  [X]      │
└────────────────────────────────────┘
```

- **마이크 아이콘**: 음성 인식 상태 표시
- **일시 중지**: 임시 중단 (다시 누르면 재개)
- **설정**: 음성 입력 옵션
- **X**: 음성 입력 종료

### 음성 명령어

| 명령어 | 동작 |
|--------|------|
| "마침표" | . 입력 |
| "쉼표" | , 입력 |
| "물음표" | ? 입력 |
| "새 줄" 또는 "줄 바꿈" | Enter 키 |
| "삭제" | 마지막 단어 삭제 |
| "전체 삭제" | 전체 삭제 |

## 설정 방법

### 마이크 설정 확인

1. **설정** → **시스템** → **소리**
2. **입력** 섹션에서 마이크 선택
3. **마이크 테스트**로 동작 확인

### 음성 인식 언어 설정

1. **설정** → **시간 및 언어** → **음성**
2. **음성 언어** 선택 (한국어)
3. **마이크** 설정 확인

### 개인 정보 설정

1. **설정** → **개인 정보 및 보안** → **음성**
2. **온라인 음성 인식** 활성화 (더 정확한 인식)

## 정확도 향상 팁

### 1. 발음 명확하게

```
✅ 좋은 예: "회원가입 API 만들어줘"
❌ 나쁜 예: "회원가입에이피아이만들어줘" (빠르게 붙여 말하면 인식 오류)
```

### 2. 적당한 속도 유지

- 너무 빠르면 인식 오류
- 너무 느리면 단어 분리 오류
- **평소 대화 속도**가 가장 좋음

### 3. 주변 소음 최소화

- 에어컨, 선풍기 소음 주의
- 다른 사람 목소리가 섞이지 않게
- 헤드셋 마이크 권장

### 4. 기술 용어 주의

```
일부 기술 용어는 잘못 인식될 수 있음:

말한 것 → 인식된 것
"API" → "에이피아이" 또는 "API"
"Supabase" → "수파베이스" 또는 "슈퍼베이스"
"npm" → "엔피엠" 또는 "NPM"
```

**팁**: 영어 단어는 발음 그대로, 또는 한글로 풀어서 말하면 인식률 향상

## Windows 음성 입력의 장단점

| 장점 | 단점 |
|------|------|
| 추가 설치 불필요 | 인터넷 연결 필요 (온라인 인식 시) |
| 빠른 실행 (Win+H) | 기술 용어 인식 오류 |
| 무료 사용 | 영어 혼용 문장 어려움 |
| 실시간 변환 | 주변 소음에 민감 |

## 다른 음성 입력 방법

Windows 기본 음성 입력 외에도 **ChatGPT, Gemini 등 AI 서비스의 음성 입력**을 활용하는 방법도 있습니다. AI 서비스는 기술 용어 인식률이 더 높고, 영어 혼용 문장도 잘 처리합니다.

> **참고**: "음성으로 지시하기 - AI 활용" 팁을 확인하세요.

## 주의사항

- 마이크가 없으면 사용 불가
- 인터넷 연결 시 더 정확한 인식 (온라인 음성 인식)
- 공공장소에서는 사용 어려움
- 민감한 정보는 음성 입력 자제 (보안 고려)
