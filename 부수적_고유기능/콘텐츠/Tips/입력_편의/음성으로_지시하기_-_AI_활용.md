# 음성으로 지시하기 - AI 활용

## 핵심 요약

ChatGPT나 Gemini 웹사이트에서 음성 입력 기능을 사용한 후, 인식된 텍스트를 복사해서 Claude Code에 붙여넣는 방법입니다. Windows 기본 음성 입력보다 기술 용어 인식률이 높고, 영어 혼용 문장도 잘 처리합니다.

## 왜 AI 서비스를 활용하나?

### Windows 기본 음성 입력의 한계

```
말한 것:
"사용자 테이블에서 이메일이 at 지메일 닷컴으로 끝나는 사람들 찾아줘"

Windows 인식 결과:
"사용자 테이블에서 이메일이 앳 지메일 닷 컴으로 끝나는 사람들 찾아줘"
```

### AI 서비스의 장점

```
같은 말을 AI 서비스에서:

ChatGPT/Gemini 인식 결과:
"사용자 테이블에서 이메일이 @gmail.com으로 끝나는 사람들 찾아줘"
```

**AI가 문맥을 이해하고 적절한 기호로 변환해줍니다.**

## 방법별 비교

| 방법 | 기술 용어 인식 | 영어 혼용 | 속도 |
|------|---------------|----------|------|
| Windows 기본 (Win+H) | 보통 | 어려움 | 빠름 |
| ChatGPT 음성 | 높음 | 좋음 | 보통 |
| Gemini 음성 | 높음 | 좋음 | 보통 |

## ChatGPT 음성 입력 활용

### 사용법

1. 웹 브라우저에서 **chat.openai.com** 접속
2. 입력창 오른쪽의 **마이크 아이콘** 클릭
3. 음성으로 지시사항 말하기
4. 인식된 텍스트 확인
5. **텍스트 선택 → 복사 (Ctrl+C)**
6. Claude Code 터미널에 **붙여넣기 (Ctrl+V)**

### ChatGPT 음성의 장점

```
입력:
"함수 만들어줘 파라미터는 유저 아이디랑 이메일이고
리턴은 불리언으로 해줘"

ChatGPT가 인식한 텍스트:
"함수 만들어줘. 파라미터는 userId와 email이고,
리턴은 boolean으로 해줘."
```

- 영어 단어 자동 변환 (userId, email, boolean)
- 문장 부호 자동 삽입
- 개발 용어 인식률 높음

## Google Gemini 음성 입력 활용

### 사용법

1. 웹 브라우저에서 **gemini.google.com** 접속
2. 입력창의 **마이크 아이콘** 클릭
3. 음성으로 말하기
4. 인식된 텍스트 확인 및 복사
5. Claude Code에 붙여넣기

### Gemini의 장점

- Google 계정만 있으면 무료
- 한국어 인식 정확도 높음
- 코드 관련 용어 잘 인식

## 실전 워크플로우

### 예시: API 함수 요청

```
1. ChatGPT 또는 Gemini 웹사이트 열기

2. 마이크 아이콘 클릭

3. 말하기:
   "로그인 API 만들어줘
    이메일이랑 패스워드 받아서
    Supabase로 인증하고
    JWT 토큰 반환하는 거야"

4. 인식된 텍스트 확인:
   "로그인 API 만들어줘.
    email과 password를 받아서
    Supabase로 인증하고
    JWT 토큰을 반환하는 거야."

5. 텍스트 복사 (Ctrl+C)

6. Claude Code에 붙여넣기 (Ctrl+V)
```

**소요 시간**: 약 30초 (타이핑 대비 절반 이하)

| 코드 용어 | 매우 좋음 | 좋음 |
| 한국어 | 좋음 | 매우 좋음 |

| 상황 | 추천 방법 |

| 일반적인 긴 지시사항 | Gemini 음성 |

## 팁: 효율적인 사용법

### 브라우저 탭 미리 열어두기

```
작업 시작 전:
- 탭 1: Claude Code가 실행된 터미널
- 탭 2: ChatGPT 또는 Gemini (음성 입력용)

Alt+Tab으로 빠르게 전환하며 사용
```

### 긴 지시사항 나눠서 입력

```
한 번에 길게 말하기보다 문장 단위로 끊어서:

1차: "로그인 페이지 만들어줘"
     → 복사 → 붙여넣기

2차: "이메일 입력칸이랑 비밀번호 입력칸 추가하고"
     → 복사 → 붙여넣기

3차: "로그인 버튼 누르면 Supabase 인증 호출해줘"
     → 복사 → 붙여넣기
```

## 주의사항

- 음성 데이터가 서버로 전송됨 (개인정보 주의)
- 민감한 비즈니스 정보는 음성 입력 자제
- 인터넷 연결 필수
- 인식 오류 확인 후 붙여넣기 (특히 숫자, 이메일 주소)
- ChatGPT/Gemini 계정 로그인 필요
