# SSAL Works Dev Package란 무엇인가

> AI와 협업하여 웹 프로젝트를 완성하는 올인원 개발 패키지

---

## 패키지의 정체성

SSAL Works Dev Package는 Claude Code와 함께 웹 프로젝트를 처음부터 끝까지 수행할 수 있도록 설계된 통합 개발 환경입니다. 다운로드하고 압축을 풀면, 프로젝트에 필요한 모든 것이 준비되어 있습니다.

일반적으로 새 프로젝트를 시작하려면 폴더 구조를 설계하고, 규칙을 정하고, 자동화 스크립트를 만들어야 합니다. 이 과정만 해도 몇 주가 걸릴 수 있습니다. SSAL Works Dev Package는 이 모든 것을 미리 준비해 두었습니다.

---

## 무엇이 들어있는가

패키지에는 크게 네 가지가 포함되어 있습니다.

첫째, 프로젝트 디렉토리 구조입니다. P0부터 S5까지 10단계로 나뉜 폴더 구조가 준비되어 있습니다. P 단계는 기획 과정이고, S 단계는 개발 과정입니다. 어떤 파일을 어디에 저장해야 하는지 고민할 필요가 없습니다.

둘째, AI 협업 설정입니다. Claude Code가 프로젝트 규칙을 자동으로 인식하도록 CLAUDE.md 파일이 준비되어 있습니다. 7대 작업 규칙, 서브에이전트 설정, 스킬 설정, 슬래시 명령어까지 모두 포함되어 있습니다.

셋째, Order Sheet 템플릿입니다. Claude Code에게 작업을 지시할 때 사용하는 표준 양식이 32개 준비되어 있습니다. 기획 단계부터 개발 완료까지, 모든 단계에 맞는 템플릿이 있습니다.

넷째, 자동화 스크립트입니다. 코드를 저장하면 자동으로 배포 폴더에 복사되는 Pre-commit Hook이 설정되어 있습니다. 수작업으로 파일을 복사할 필요가 없습니다.

---

## 왜 필요한가

Claude Code는 강력한 도구입니다. 하지만 체계적인 프로젝트 관리 없이 사용하면 문제가 생깁니다. 파일이 여기저기 흩어지고, 어떤 작업을 했는지 추적하기 어려워집니다. 나중에 수정이 필요할 때 어디서부터 손대야 할지 모르게 됩니다.

SSALWorks 방법론은 이 문제를 해결합니다. 모든 파일은 정해진 위치에 저장됩니다. 모든 작업은 기록됩니다. AI가 실수하지 않도록 규칙이 명확히 정의되어 있습니다. Dev Package는 이 방법론을 바로 사용할 수 있게 패키지로 만든 것입니다.

---

## 어떻게 사용하는가

사용법은 간단합니다. 패키지를 다운로드하고 압축을 풉니다. 폴더 이름을 자신의 프로젝트명으로 바꿉니다. 터미널에서 Pre-commit Hook 설치 스크립트를 한 번 실행합니다. 이제 Claude Code를 실행하면 모든 준비가 완료됩니다.

Claude Code는 CLAUDE.md를 읽고 프로젝트 규칙을 자동으로 인식합니다. 파일 저장 위치, 명명 규칙, 검증 절차를 모두 알고 있습니다. 사용자는 Order Sheet를 선택해서 원하는 작업을 지시하기만 하면 됩니다.

---

## 누구를 위한 것인가

이 패키지는 비개발자에게 특히 유용합니다. 코딩을 몰라도 AI와 함께 웹 서비스를 만들 수 있습니다. 프로세스를 따라가면 됩니다.

개발자에게도 유용합니다. 프로젝트 초기 설정에 드는 시간을 절약할 수 있습니다. 검증된 구조를 바로 사용할 수 있습니다.

팀에게도 유용합니다. 표준화된 구조 덕분에 누가 작업해도 일관된 결과물이 나옵니다. 인수인계가 쉬워집니다.

---

## 결론

SSAL Works Dev Package는 단순한 템플릿이 아닙니다. 프로젝트 구조, AI 협업 규칙, 작업 지시 템플릿, 자동화 스크립트가 하나로 통합된 개발 환경입니다.

Claude Code만으로는 체계적인 프로젝트 관리가 어렵습니다. SSALWorks Dev Package를 사용하면 처음부터 올바른 구조로 시작할 수 있습니다. 다운로드하고, 압축 풀고, 시작하세요.
