# 디렉터리 구조 문서 만들기

## 요약
`P0_프로젝트_구조/Project_Directory_Structure.md`에 전체 폴더 구조 문서화. 파일 저장 위치 명확해지고, Claude Code도 올바른 위치에 파일 저장.

## 상세

### 문서에 포함할 내용

| 섹션 | 내용 |
|------|------|
| 전체 폴더 트리 | 한눈에 보이는 구조도 |
| 각 폴더 설명 | 용도와 저장할 파일 유형 |
| 파일 저장 규칙 | 어떤 파일을 어디에 저장 |

### 파일 저장 규칙 예시

| 파일 유형 | 저장 위치 |
|----------|----------|
| HTML 페이지 | S?_*/Frontend/ + Production/Frontend/ |
| API 코드 | S?_*/Backend_API/ + Production/Backend_API/ |
| SQL 파일 | S?_*/Database/ |
| 테스트 코드 | S?_*/Testing/ |

### 활용 팁
- CLAUDE.md에서 참조 설정
- 폴더 변경 시 즉시 문서 업데이트
- 버전과 업데이트 날짜 기록
