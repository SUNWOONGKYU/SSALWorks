# 프로젝트 진행 상황 문서 만들기

## 요약
`P0_프로젝트_구조/Project_Status.md`에 현재 진행 상황 기록. 어디까지 완료됐고 다음에 무엇을 해야 하는지 파악 가능.

## 상세

### 문서에 포함할 내용

| 섹션 | 내용 |
|------|------|
| 전체 진행률 | 단계별 완료 현황 |
| 완료된 단계 | 끝난 단계와 주요 산출물 |
| 현재 진행 중 | 작업 중인 Task 목록 |
| 다음 작업 | 앞으로 해야 할 일 |
| 블로커/이슈 | 막힌 부분 |

### 상태 표시

```
✅ 완료
🔄 진행중
⏳ 대기
⚠️ 블로커
```

### Project_Status vs work_logs

| 항목 | Project_Status.md | work_logs/current.md |
|------|-------------------|---------------------|
| 목적 | 전체 진행 현황 | 상세 작업 기록 |
| 범위 | 프로젝트 전체 | 최근 세션 |
| 상세도 | 요약 | 상세 |
| 대상 | 전체 팀원 | 개발자/AI |

둘 다 함께 사용하면 가장 효과적.
