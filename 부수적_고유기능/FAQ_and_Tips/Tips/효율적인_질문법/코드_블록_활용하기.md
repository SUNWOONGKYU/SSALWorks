# 코드 블록 활용하기

## 💡 Tip

코드를 공유할 때는 **백틱 3개 (```)** 로 감싸서 코드 블록으로 만드세요.

## 기본 사용법

````markdown
```
여기에 코드 작성
```
````

## 언어 지정하기

언어를 지정하면 **문법 강조(Syntax Highlighting)**가 적용됩니다.

````markdown
```javascript
function hello() {
    console.log('Hello, World!');
}
```
````

## 자주 사용하는 언어 태그

| 언어 | 태그 |
|------|------|
| JavaScript | `javascript` 또는 `js` |
| TypeScript | `typescript` 또는 `ts` |
| HTML | `html` |
| CSS | `css` |
| SQL | `sql` |
| JSON | `json` |
| Bash/Shell | `bash` 또는 `sh` |
| Markdown | `markdown` 또는 `md` |

## 인라인 코드

한 줄 코드는 백틱 1개로:

```markdown
`const x = 10;` 처럼 사용합니다.
```

결과: `const x = 10;` 처럼 사용합니다.

## 파일 경로 표시

```markdown
파일 위치: `Production/Frontend/index.html`
```

## 실제 예시

"다음 코드에서 오류가 발생합니다:"

```javascript
// Production/Frontend/index.html:612
const { data, error } = await supabaseClient.auth.signInWithOAuth({
    provider: 'google',
    options: {
        redirectTo: '/wrong/path/index.html'  // 여기가 문제!
    }
});
```

## 장점

1. **가독성 향상** - 코드와 일반 텍스트 구분
2. **복사 용이** - 코드 블록에서 바로 복사
3. **문법 강조** - 키워드, 문자열 등 색상 구분
4. **줄바꿈 유지** - 코드 구조 그대로 유지
