# 로그인 후 닉네임이 표시되지 않아요

## 질문
Google 로그인은 성공했는데 헤더에 닉네임이 표시되지 않아요.

## 답변

### 원인
1. users 테이블에 닉네임이 저장되지 않음
2. 닉네임 조회 로직이 없음
3. UI에 닉네임 표시 요소가 없음

### 해결 방법

**1. 닉네임 조회 로직 추가 (JavaScript)**
```javascript
async function checkAuthStatus() {
    const { data: { session } } = await supabaseClient.auth.getSession();

    if (session && session.user) {
        // 사용자 닉네임 가져오기
        const { data: userData } = await supabaseClient
            .from('users')
            .select('nickname')
            .eq('id', session.user.id)
            .single();

        let nickname = '사용자';
        if (userData?.nickname) {
            nickname = userData.nickname;
        } else if (session.user.user_metadata?.nickname) {
            nickname = session.user.user_metadata.nickname;
        } else if (session.user.email) {
            nickname = session.user.email.split('@')[0];
        }

        showLoggedInUI(nickname);
    }
}
```

**2. HTML에 닉네임 표시 요소 추가**
```html
<div id="headerUserBtns" style="display: none;">
    <span id="userNickname"></span>
    <button onclick="logout()">로그아웃</button>
</div>
```

### 닉네임 우선순위
1. users 테이블의 nickname
2. Google user_metadata의 nickname
3. 이메일 @ 앞부분

### 관련 파일
- `Production/Frontend/index.html` - checkAuthStatus() 함수
