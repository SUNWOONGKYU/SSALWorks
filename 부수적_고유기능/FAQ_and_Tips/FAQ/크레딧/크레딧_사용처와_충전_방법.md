# 크레딧 사용처와 충전 방법

## 질문
크레딧은 어디에 사용되고 어떻게 충전하나요?

## 답변

### 크레딧 사용처

| 기능 | 크레딧 소모 |
|------|-----------|
| AI 질문 (기본) | 1 크레딧/질문 |
| AI 질문 (복잡한 코드 생성) | 3~5 크레딧/질문 |
| 문서 자동화 | 2 크레딧/문서 |
| 코드 리뷰 | 2 크레딧/리뷰 |

### 플랜별 크레딧

| 플랜 | 월간 크레딧 | 추가 혜택 |
|------|-----------|----------|
| Basic (무료) | 10 크레딧 | - |
| Pro (₩19,900) | 무제한 | 우선 지원 |
| Enterprise | 무제한 | 전용 지원 |

### 크레딧 충전 방법

1. **마이페이지 > 크레딧 관리** 접속
2. 원하는 충전 금액 선택
3. 카드 결제 진행

### 크레딧 유효기간

- **충전 크레딧**: 충전일로부터 1년
- **월간 크레딧**: 해당 월 내 사용 (이월 불가)

### 크레딧 사용 내역 확인

```
마이페이지 > 크레딧 관리 > 사용 내역
```

- 일별/월별 사용량 그래프
- 기능별 사용 내역
- 잔여 크레딧 확인
