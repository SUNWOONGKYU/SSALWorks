# P2-4 Database Design 안내문

## 개요

데이터베이스를 설계하는 작업입니다.
데이터 모델과 스키마를 정의합니다.

## 목적

- 데이터 구조 설계
- 테이블 관계 정의
- 인덱스 전략 수립
- 데이터 무결성 보장

## 주요 내용

### ERD (Entity-Relationship Diagram)
- 엔티티 정의
- 관계 설정 (1:1, 1:N, N:M)
- 속성 정의

### 스키마 설계
- 테이블 구조
- 컬럼 타입
- 제약 조건
- 인덱스

### 보안 설계
- RLS (Row Level Security)
- 권한 관리

## 예상 산출물

| 산출물 | 설명 |
|--------|------|
| ERD | 데이터 모델 다이어그램 |
| 스키마 SQL | 테이블 생성 스크립트 |
| RLS 정책 | 보안 정책 SQL |
