---

## 작업 내용

### 1. 컬러 시스템
- Primary/Secondary/Accent 컬러 정의
- Semantic 컬러 (Success/Warning/Error/Info)
- Neutral 컬러 (Background/Text/Border)

### 2. 타이포그래피
- 폰트 패밀리 선정
- 폰트 사이즈 체계 (H1~H6, Body, Caption)
- 라인 높이 및 자간

### 3. 컴포넌트 스타일
- 버튼 스타일 (Primary/Secondary/Outline)
- 입력 필드 스타일 (Default/Focus/Error)
- 카드, 모달 등 공통 컴포넌트

### 4. 아이콘 & 이미지
- 아이콘 라이브러리 선정
- 이미지 스타일 가이드
- 아이콘 사이즈 규격
