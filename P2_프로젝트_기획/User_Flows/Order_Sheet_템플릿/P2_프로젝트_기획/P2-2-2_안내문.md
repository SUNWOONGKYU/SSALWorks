# P2-2-2 User Flows 안내문

## 개요

사용자 흐름(User Flow)을 설계하는 작업입니다.
사용자가 서비스를 이용하는 경로를 시각화합니다.

## 목적

- 사용자 경험 설계
- 화면 간 이동 경로 정의
- 예외 상황 처리 설계
- 개발 가이드 제공

## 주요 내용

### User Flow 다이어그램
- 회원가입/로그인 흐름
- 핵심 기능 사용 흐름
- 결제/구독 흐름
- 오류 처리 흐름

### 시나리오
- Happy Path (정상 흐름)
- Edge Cases (예외 상황)

## 예상 산출물

|--------|------|
