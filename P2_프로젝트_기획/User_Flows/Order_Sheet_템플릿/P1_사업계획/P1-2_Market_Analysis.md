# Order Sheet - P1-2 시장 분석

## 작업 지시

**Claude AI에게**: 이 프로젝트의 타겟 시장을 분석하고 경쟁 환경을 조사해주세요.

---

## 작업 내용

### 1. 타겟 시장 정의
- 어떤 시장을 타겟으로 하는가?
- 시장 규모 (TAM/SAM/SOM 분석)
- 시장 성장률 및 트렌드

### 2. 경쟁사 분석
- 주요 경쟁사 3~5개 선정
- 각 경쟁사의 강점/약점 분석
- 시장 포지셔닝 비교

### 3. SWOT 분석
- Strengths (강점)
- Weaknesses (약점)
- Opportunities (기회)
- Threats (위협)

### 4. 시장 진입 전략
- 차별화 포인트
- 초기 타겟 세그먼트
- Go-to-Market 전략 제안
