# P1-2 Market Analysis 안내문

## 개요

시장 분석을 수행하여 사업 기회와 경쟁 환경을 파악하는 작업입니다.
데이터 기반의 시장 이해를 통해 전략적 의사결정을 지원합니다.

## 목적

- 목표 시장의 규모와 성장성 파악
- 경쟁사 분석 및 차별화 전략 수립
- 고객 세그먼트 정의
- 시장 진입 전략 기반 마련

## 주요 내용

### 시장 규모 분석
- TAM (Total Addressable Market)
- SAM (Serviceable Available Market)
- SOM (Serviceable Obtainable Market)

### 경쟁사 분석
- 주요 경쟁사 식별
- 강점/약점 분석
- 시장 포지셔닝

### 고객 분석
- 목표 고객 정의
- 페르소나 작성
- 고객 니즈 파악

## 예상 산출물

| 산출물 | 설명 |
|--------|------|
| 시장 분석 보고서 | 시장 규모, 트렌드 |
| 경쟁사 분석표 | 경쟁사별 비교 |
| 고객 페르소나 | 목표 고객 프로필 |

## 실행 조건

- **실행 유형**: Human-AI (리서치 + AI 분석)
- **의존성**: P1-1 완료
- **필요 정보**: 산업 분야, 타겟 시장, 경쟁사 정보
