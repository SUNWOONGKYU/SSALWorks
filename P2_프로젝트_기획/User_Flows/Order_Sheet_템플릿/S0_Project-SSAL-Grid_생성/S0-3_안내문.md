# S0-3 SSAL Grid Supabase 안내문

## 개요

SSAL Grid의 Supabase 연동을 설정하는 작업입니다.
DB 테이블과 동기화 스크립트를 구축합니다.

## 목적

- Grid 데이터 저장소 구축
- 실시간 진행 상황 조회
- 팀 협업 지원
- 자동화된 데이터 관리

## 주요 내용

### Supabase 설정
- ssalworks_tasks 테이블 생성
- stage_verification 테이블 생성
- RLS 정책 설정

### 동기화 스크립트
- JSON → DB 동기화
- DB → JSON 동기화
- 자동 업데이트

## 예상 산출물

| 산출물 | 설명 |
|--------|------|
| schema.sql | 테이블 스키마 |
| sync_task_results_to_db.js | 동기화 스크립트 |
| generate_result_json_from_db.js | 역동기화 스크립트 |
