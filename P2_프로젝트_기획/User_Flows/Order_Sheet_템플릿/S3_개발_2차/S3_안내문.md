# S3 개발 2차 (Advanced Features) 안내문

## 개요

고급 기능을 개발하는 단계입니다.
AI 연동, 외부 서비스 통합 등 확장 기능을 구현합니다.

## 목적

- 고급 기능 구현
- AI 연동
- 외부 서비스 통합
- 사용자 경험 향상

## Stage 범위

### Area별 작업
| Area | 내용 |
|------|------|
| BI (Backend Infra) | AI 인프라 |
| BA (Backend APIs) | 고급 API |
| S (Security) | 보안 강화 |
| E (External) | 외부 연동 |

## 실행 방식

- **SSAL Grid 기반**: ssalworks_tasks 테이블에서 Stage=3인 Task 실행
- **서브에이전트 투입**: Task별 지정된 agent 사용
- **검증 필수**: Verification Agent로 검증 후 완료

## 완료 조건

- 모든 S3 Task 완료
- Stage Gate 검증 통과
- PO 승인
