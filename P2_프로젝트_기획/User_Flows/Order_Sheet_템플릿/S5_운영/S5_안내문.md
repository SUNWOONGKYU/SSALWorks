# S5 운영 (Operations) 안내문

## 개요

프로덕션 배포와 운영을 수행하는 단계입니다.
실제 서비스 런칭과 운영 체계를 구축합니다.

## 목적

- 프로덕션 배포
- 운영 체계 구축
- 모니터링 시스템
- 유지보수 프로세스

## Stage 범위

### Area별 작업
| Area | 내용 |
|------|------|
| M (Documentation) | 운영 문서화 |
| F (Frontend) | 프로덕션 빌드 |
| BA (Backend APIs) | API 운영 |
| D (Database) | DB 운영/백업 |
| S (Security) | 보안 모니터링 |
| O (DevOps) | 인프라 운영 |

## 실행 방식

- **SSAL Grid 기반**: ssalworks_tasks 테이블에서 Stage=5인 Task 실행
- **서브에이전트 투입**: Task별 지정된 agent 사용
- **검증 필수**: Verification Agent로 검증 후 완료

## 완료 조건

- 모든 S5 Task 완료
- Stage Gate 검증 통과
- PO 승인
- 서비스 정상 운영
