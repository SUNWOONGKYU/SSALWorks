📦 프로젝트 작업 안내

선택한 프로젝트에서 작업을 진행합니다.

프로젝트 작업 흐름

  1. 좌측 진행 프로세스에서 현재 단계 확인
  2. 해당 단계의 Order Sheet 작성
  3. Inbox에 발행하여 Claude Code에 작업 요청
  4. Outbox에서 작업 결과 확인

Order Sheet 작성 방법

  • 요청 유형을 선택하세요
  • 상세 요청사항을 구체적으로 작성하세요
  • 필요시 참고 자료를 첨부하세요
⚠️ 주의: Order Sheet는 명확하고 구체적으로 작성할수록 더 정확한 결과를 얻을 수 있습니다.
