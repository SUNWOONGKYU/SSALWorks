📋 SP-1 디렉토리 구조 자동 생성 안내

이 단계에서 수행하는 작업

프로젝트 디렉토리 구조 자동 생성을 위한 Order Sheet를 발행합니다.

작업 목적

새 프로젝트를 위한 표준 디렉토리 구조(P1~P3, S1~S5)를 자동 생성합니다.

주요 작업 내용

  • P1~P3 Preliminary 단계 폴더 생성
  • S1~S5 Stage 단계 폴더 생성
  • 기본 문서 파일 생성
  • 설정 파일 생성

필요한 입력 정보

  • 프로젝트명
  • 저장 경로
  • 커스텀 폴더 요구사항 (선택)

예상 결과물

  • 전체 디렉토리 구조
  • PROJECT_DIRECTORY_STRUCTURE.md
  • PROJECT_STATUS.md
  • README.md

확인 버튼을 클릭하면 Workspace에 Order Sheet가 로드됩니다.
