주요 작업 내용

  • Phase 설정 (1~4)
  • Area 설정 (FE, BA, AU, DB, TS, DP)
  • Task 목록 정의
  • Task ID 규칙 적용

필요한 입력 정보

  • 프로젝트명
  • Phase별 범위 정의
  • 초기 Task 목록

예상 결과물

  • project_grid.json
  • Task Instruction 템플릿
  • Phase별 Task 목록
