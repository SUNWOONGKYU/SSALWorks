# 📋 SP-2 SAL GRID 생성 안내

## 이 단계에서 수행하는 작업

**프로젝트 그리드(SAL GRID) 생성**을 위한 Order Sheet를 발행합니다.

### 작업 목적
Task Instruction 기반 개발을 위한 프로젝트 그리드 시스템을 생성합니다.

### 주요 작업 내용
- Phase 설정 (1~4)
- Area 설정 (FE, BA, AU, DB, TS, DP)
- Task 목록 정의
- Task ID 규칙 적용

### 필요한 입력 정보
- 프로젝트명
- Phase별 범위 정의
- 초기 Task 목록

### 예상 결과물
- project_grid.json
- Task Instruction 템플릿
- Phase별 Task 목록

---

**확인 버튼을 클릭하면 Workspace에 Order Sheet가 로드됩니다.**
