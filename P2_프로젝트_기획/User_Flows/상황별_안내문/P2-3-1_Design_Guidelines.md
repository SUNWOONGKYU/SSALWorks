## 2. 이 단계에서 수행하는 작업

**UI/UX 디자인 가이드라인 작성**을 위한 Order Sheet를 발행합니다.

---

## 3. 작업 목적

프로젝트 전체에서 일관된 디자인을 유지하기 위한 가이드라인을 정의합니다.

---

## 4. 주요 작업 내용

### 3.1 컬러 시스템
- Primary/Secondary/Accent 컬러
- Semantic 컬러 (Success/Warning/Error)
- Neutral 컬러 (Background/Text/Border)

### 3.2 타이포그래피
- 폰트 패밀리 선정
- 폰트 사이즈 체계 (H1~H6, Body)
- 라인 높이 및 자간

### 3.3 컴포넌트 스타일
- 버튼 스타일 (Primary/Secondary/Outline)
- 입력 필드 스타일
- 카드, 모달 등 공통 컴포넌트

### 3.4 레이아웃 시스템
- 그리드 시스템
- 브레이크포인트 정의
- 간격(Spacing) 체계
