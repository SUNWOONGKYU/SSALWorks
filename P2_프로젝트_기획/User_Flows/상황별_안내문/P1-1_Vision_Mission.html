2. 이 단계에서 수행하는 작업

비전/미션 문서 작성을 위한 Order Sheet를 발행합니다.

3. 작업 목적

프로젝트의 핵심 방향을 정의하는 비전(Vision)과 미션(Mission) 문서를 작성합니다.

4. 주요 작업 내용

4.1 비전 정의

  • 프로젝트의 장기적 목표
  • 5년 후, 10년 후 달성할 상태
  • 영감을 주는 방향성

4.2 미션 정의

  • 프로젝트의 존재 이유
  • 핵심 활동 영역
  • 가치 제공 방식

4.3 핵심 가치 설정

  • 조직/서비스의 근본 가치
  • 의사결정 기준
  • 행동 원칙

4.4 성공 기준 수립

  • 측정 가능한 목표
  • KPI 정의
  • 마일스톤 설정

5. 필요한 입력 정보

| 항목 | 설명 |

|------|------|

| 프로젝트명 | 서비스/프로젝트 이름 |

| 프로젝트 개요 | 무엇을 만드는 프로젝트인지 |

| 목표 고객/사용자 | 주요 타겟 사용자 |

| 기대 효과 | 프로젝트로 달성할 효과 |
