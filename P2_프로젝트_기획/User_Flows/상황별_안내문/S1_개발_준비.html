📋 S1 개발 준비 단계 안내

이 단계에서 수행하는 작업

본격적인 개발 전 준비 작업을 위한 Order Sheet를 발행합니다.

작업 목적

개발 환경을 구축하고 개발 시작을 위한 모든 준비를 완료합니다.

주요 작업 내용

  • 개발 환경 설정 (IDE, 도구)
  • 버전 관리 설정 (Git)
  • 패키지 의존성 설치
  • 코드 컨벤션 설정
  • CI/CD 파이프라인 기초 설정

필요한 입력 정보

  • 기술 스택 확정
  • 팀 개발 환경 표준
  • 저장소 정보

예상 결과물

  • 개발 환경 설정 완료
  • Git 저장소 초기화
  • 프로젝트 기본 구조
  • 개발 가이드 문서

확인 버튼을 클릭하면 Workspace에 Order Sheet가 로드됩니다.
