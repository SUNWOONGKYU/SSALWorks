Project SAL Grid - Viewer

Grid Viewer 안내

Project SAL Grid 뷰어로 Task 진행 상황을 시각적으로 확인합니다.

Viewer 기능

  • 5x11 매트릭스 시각화
  • Stage별 Task 진행률 표시
  • Task 상태 (Pending/In Progress/Completed)
  • 검증 상태 확인

사용 방법

  1. Viewer 페이지 열기
  2. Supabase에서 데이터 자동 로드
  3. Stage/Area별 필터링
  4. Task 클릭하여 상세 정보 확인

관련 파일 위치

  • Project-SSAL-Grid/viewer/viewer.html

확인 버튼을 클릭하면 Grid Viewer가 열립니다.
