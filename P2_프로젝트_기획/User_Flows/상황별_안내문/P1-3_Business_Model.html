이전 항목에서 완료된 것 (P1-1, P1-2)

  • ✅ 프로젝트 비전/미션 정의
  • ✅ 시장 규모 분석
  • ✅ 경쟁사 분석
  • ✅ 타겟 고객 정의

현재 항목에서 수행할 것 (P1-3)

  • 🔄 비즈니스 모델 캔버스 작성
  • 🔄 수익 모델 정의 (구독, 광고, 수수료 등)
  • 🔄 비용 구조 분석
  • 🔄 손익 분기점 분석

다음 Stage에서 수행할 것 (P2)

  • ⏳ 프로젝트 디렉토리 구조 설계
  • ⏳ 기능 요구사항 정의
  • ⏳ UI/UX 설계
  • ⏳ DB 스키마 설계

2. 이 단계에서 수행하는 작업

비즈니스 모델 문서 작성을 위한 Order Sheet를 발행합니다.

3. 작업 목적

프로젝트의 수익 창출 방법과 비즈니스 구조를 정의합니다.

4. 주요 작업 내용

4.1 비즈니스 모델 캔버스

1. 고객 세그먼트 (Customer Segments)

2. 가치 제안 (Value Propositions)

3. 채널 (Channels)

4. 고객 관계 (Customer Relationships)

5. 수익원 (Revenue Streams)

6. 핵심 자원 (Key Resources)

7. 핵심 활동 (Key Activities)

8. 핵심 파트너 (Key Partners)

9. 비용 구조 (Cost Structure)

4.2 수익 모델 정의

  • 주요 수익원 (구독, 광고, 수수료 등)
  • 가격 정책
  • 예상 매출 시나리오

4.3 비용 구조 분석

  • 고정 비용
  • 변동 비용
  • 초기 투자 비용

4.4 손익 분기점 분석

  • BEP (Break-Even Point) 계산
  • 목표 달성 시나리오
