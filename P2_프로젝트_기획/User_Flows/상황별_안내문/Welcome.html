🌾 SSAL Works에 오신 것을 환영합니다!

SSAL Works는 비개발자가 AI(Claude Code)와 협업하여 웹사이트를 개발하는 플랫폼입니다.

🔧 Step 1: Claude Code 설치

SSAL Works를 사용하려면 먼저 Claude Code를 설치해야 합니다.

1. Node.js 설치

nodejs.org에서 LTS 버전 다운로드 → 설치

2. Claude Code 설치

npm install -g @anthropic-ai/claude-code

3. 설치 확인

claude --version
💡 Tip: 자세한 설치 방법과 오류 해결은 학습 콘텐츠 > Tips > 설치/실행을 참고하세요.

🚀 Step 2: 프로젝트 시작

  1. 프로젝트 등록 - 새 프로젝트를 등록하세요
  2. 진행 프로세스 - P1 사업계획부터 단계별 진행
  3. Order Sheet - AI에게 작업을 요청하세요
📘 처음이신가요? 예시 프로젝트를 먼저 살펴보시면 진행 방법을 쉽게 이해할 수 있습니다.
