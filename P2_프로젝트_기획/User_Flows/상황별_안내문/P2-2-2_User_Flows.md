- 핵심 기능 플로우
- 결제 플로우 (해당 시)

### 3.2 플로우 설계 요소
- 시작점/종료점
- 단계별 액션
- 분기 조건
- 예외 처리

### 3.3 플로우 다이어그램
```
[시작] → [단계1] → [분기] → [단계2A/2B] → [종료]
```
