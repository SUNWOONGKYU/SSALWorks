# 📋 P2-2-2 User Flow 설계 안내

## 이 단계에서 수행하는 작업

**사용자 흐름(User Flow) 문서 작성**을 위한 Order Sheet를 발행합니다.

### 작업 목적
사용자가 서비스를 이용하는 과정을 단계별로 설계합니다.

### 주요 작업 내용
- 주요 사용자 시나리오 정의
- 각 시나리오별 User Flow 다이어그램
- 분기점 및 예외 처리 정의
- 사용자 여정 맵 작성

### 필요한 입력 정보
- 주요 기능 목록
- 사용자 유형 (페르소나)
- 핵심 시나리오

### 예상 결과물
- User_Flows.md 문서
- User Flow 다이어그램
- 사용자 여정 맵
