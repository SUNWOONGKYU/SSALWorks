# 📋 P1-1 비전/미션 설정 안내

## 이 단계에서 수행하는 작업

**비전/미션 문서 작성**을 위한 Order Sheet를 발행합니다.

### 작업 목적
프로젝트의 핵심 방향을 정의하는 비전(Vision)과 미션(Mission) 문서를 작성합니다.

### 주요 작업 내용
- 프로젝트의 장기적 목표(비전) 정의
- 프로젝트의 존재 이유(미션) 정의
- 핵심 가치 설정
- 성공 기준 수립

### 필요한 입력 정보
- 프로젝트명
- 프로젝트 개요
- 목표 고객/사용자
- 기대 효과

### 예상 결과물
- Vision_Mission.md 문서
- 프로젝트 핵심 가치 정의서

---

**확인 버튼을 클릭하면 Workspace에 Order Sheet가 로드됩니다.**
