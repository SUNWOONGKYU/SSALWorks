📂 SSAL Works 예시 프로젝트

이것은 완성된 프로젝트의 샘플 파일들을 구경할 수 있는 예시 프로젝트입니다.

예시 프로젝트란?

실제 SSAL Works로 완성된 프로젝트가 어떤 구조와 파일들로 이루어져 있는지 미리 살펴볼 수 있습니다.

구경할 수 있는 내용

  • 각 단계별 폴더 구조
  • 작성된 문서 샘플 (기획서, 요구사항 등)
  • 완성된 코드 예시
  • Order Sheet 작성 예시

사용 방법

예시 프로젝트의 폴더와 파일들을 열어보며 실제 프로젝트가 어떻게 구성되는지 참고하세요.

💡 참고: 예시 프로젝트는 구경용입니다. 실제 작업을 시작하려면 "새로운 Project 추가" 버튼을 클릭하세요.
