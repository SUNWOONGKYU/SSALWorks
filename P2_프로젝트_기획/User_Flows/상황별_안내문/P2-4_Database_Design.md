---

## 2. 작업 목적

프로젝트에서 사용할 데이터베이스 구조를 설계합니다.

---

## 3. 주요 작업 내용

### 3.1 엔티티 정의
- 주요 엔티티 식별
- 엔티티별 속성 정의
- 관계 설정 (1:1, 1:N, N:M)

### 3.2 테이블 스키마 설계
- 컬럼 정의 (이름, 타입, 제약조건)
- Primary Key / Foreign Key
- 인덱스 전략

### 3.3 ERD 작성
- 엔티티 관계 다이어그램
- 관계 유형 및 카디널리티 표시

### 3.4 SQL 스크립트 생성
- 테이블 생성 SQL
- RLS 정책 (Supabase 사용 시)
- 초기 데이터 시드
