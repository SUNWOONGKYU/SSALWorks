2. 이 단계에서 수행하는 작업

3. 작업 목적

4. 주요 작업 내용

3.1 목업 대상 화면 선정

  • 핵심 화면 목록 정의
  • 우선순위 결정
  • 화면 간 관계 파악

3.2 화면별 목업 제작

  • HTML/CSS 기반 정적 목업
  • 디자인 가이드라인 적용
  • 반응형 레이아웃 고려

3.3 와이어프레임 문서화

  • 각 화면의 레이아웃 구조
  • 주요 요소 배치
  • 사용자 액션 포인트
