이 단계에서 수행하는 작업

UI 목업(Mockup) 제작을 위한 Order Sheet를 발행합니다.

작업 목적

실제 구현 전에 화면 디자인을 시각화하여 검토합니다.

주요 작업 내용

  • 주요 화면 목업 디자인
  • 레이아웃 구성
  • 컴포넌트 배치
  • 반응형 디자인 고려

필요한 입력 정보

  • 화면 목록
  • 디자인 가이드라인
  • 참고 디자인

예상 결과물

  • 화면별 목업 이미지/파일
  • Mockup 설명 문서

확인 버튼을 클릭하면 Workspace에 Order Sheet가 로드됩니다.
