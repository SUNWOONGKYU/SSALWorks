# 📋 P1-2 시장 분석 안내

## 이 단계에서 수행하는 작업

**시장 분석 문서 작성**을 위한 Order Sheet를 발행합니다.

### 작업 목적
프로젝트가 진입할 시장의 현황, 경쟁 환경, 기회 요인을 분석합니다.

### 주요 작업 내용
- 타겟 시장 규모 및 성장률 분석
- 경쟁사 분석 (SWOT)
- 시장 기회 및 위협 요인 파악
- 차별화 전략 수립

### 필요한 입력 정보
- 타겟 시장 정의
- 주요 경쟁사 목록
- 시장 데이터 (규모, 성장률 등)

### 예상 결과물
- Market_Analysis.md 문서
- 경쟁사 분석 보고서
- SWOT 분석표
