## 5. 데이터베이스 필드

### users 테이블

| 필드명 | 타입 | 설명 |
|--------|------|------|
| `installation_fee_paid` | boolean | 빌더 계정 개설비 납부 여부 |
| `installation_date` | timestamp | 개설비 납부일 |
| `subscription_status` | string | 구독 상태 (free/active) |
| `builder_id` | string | 빌더 계정 고유 ID |

### 체크 로직 예시

```javascript
// 사용자 정보 조회
const { data: userData } = await window.supabaseClient
    .from('users')
    .select('user_id, installation_fee_paid')
    .eq('id', session.user.id)
    .single();

// 빌더 계정 체크
if (!userData || !userData.installation_fee_paid) {
    alert('빌더 계정 개설비 납부 후 이용 가능합니다.');
    return;
}
```
