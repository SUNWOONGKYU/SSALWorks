# 패키지 표준 디렉토리 구조

> **버전:** 1.1
> **목적:** 다운로드 패키지에 포함될 폴더/파일 구조 정의

---

## 핵심 원칙

| 구분 | 설명 |
|------|------|
| **패키지 포함** | SSAL Works가 직접 제공하는 것 |
| **별도 설치** | SSAL Works가 제공하지 않음 (사용자가 직접 설치) |

---

## 패키지에 포함되는 것 (SSAL Works 제공)

### 1. 표준 작업 디렉토리 (P0 ~ S5)

```
├── P0_작업_디렉토리_구조_생성/
├── P1_사업계획/
├── P2_프로젝트_기획/
├── P3_프로토타입_제작/
├── S0_Project-SAL-Grid_생성/
├── S1_개발_준비/
├── S2_개발-1차/
├── S3_개발-2차/
├── S4_개발-3차/
├── S5_개발_마무리/
└── Production/
```

### 2. 개발 도구

```
├── Development_Process_Monitor/    ← 진행 상황 시각화 도구
└── Human_ClaudeCode_Bridge/        ← 작업 지시/결과 교환 시스템
```

### 3. 프로젝트 관리 파일

```
├── Project_Status.md               ← 프로젝트 진행 상태
└── Project_Directory_Structure.md  ← 폴더 구조 설명서
```

### 4. AI 설정 (선택)

```
└── .claude/                        ← Claude Code 설정 (규칙, 명령어 등)
```

### 5. 안내문/템플릿

```
└── Briefings_OrderSheets/          ← 단계별 안내문 및 Order Sheet 템플릿
```

---

## 별도 설치해야 하는 것 (SSAL Works 제공 아님)

| 도구 | 설명 | 설치 방법 |
|------|------|----------|
| **Git** | 버전 관리 시스템 | https://git-scm.com |
| **Node.js** | JavaScript 런타임 | https://nodejs.org |
| **npm** | 패키지 매니저 | Node.js에 포함 |
| **Claude Code** | AI 개발 어시스턴트 | `npm install -g @anthropic-ai/claude-code` |

> **참고:** 이것들은 외부 도구이며, SSAL Works가 제공하는 것이 아닙니다.
> 패키지 설치 후 Claude Code에게 "필수 도구 설치해줘"라고 요청하면 안내해줍니다.

## 패키지에서 제외되는 것

| 항목 | 이유 |
|------|------|
| `node_modules/` | `npm install`로 설치됨 |
| `.env` | 보안 파일, 사용자가 직접 설정 |
| `부수적_고유기능/` | SSALWorks 전용 (범용 아님) |
| 기존 작업 결과물 | 빈 템플릿만 제공 |
