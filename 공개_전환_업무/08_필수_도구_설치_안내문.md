## 4. Claude Code

AI 개발 어시스턴트입니다.

### 설치 명령어

터미널(명령 프롬프트)에서 실행:

```bash
npm install -g @anthropic-ai/claude-code
```

### 설치 확인

```bash
claude --version
```

### API 키 설정

1. https://console.anthropic.com/ 접속
2. API Keys 메뉴에서 키 생성
3. 첫 실행 시 키 입력 또는 환경변수 설정:

```bash
# Windows (PowerShell)
$env:ANTHROPIC_API_KEY="your-api-key"

# Mac/Linux
export ANTHROPIC_API_KEY="your-api-key"
```

## 7. Supabase

데이터베이스 서비스입니다.

### 가입 및 프로젝트 생성

1. https://supabase.com 접속
2. 회원가입 (GitHub 계정으로 가능)
3. New Project 생성
4. 프로젝트 Settings > API에서 키 확인

### 필요한 키

| 키 | 용도 |
|----|------|
| **Project URL** | 프로젝트 주소 |
| **anon public** | 클라이언트용 공개 키 |
| **service_role** | 서버용 비밀 키 (노출 금지) |

### 환경변수 설정

프로젝트 루트에 `.env` 파일 생성:

```
SUPABASE_URL=your-project-url
SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
```

### Supabase CLI (선택)

```bash
npm install -g supabase
```

```bash
supabase --version
```

## 문제 해결

### "command not found" 오류

- 터미널을 껐다가 다시 열어보세요
- 설치 후 시스템 재시작이 필요할 수 있습니다

### npm 권한 오류 (Mac/Linux)

```bash
sudo npm install -g @anthropic-ai/claude-code
```

### Windows에서 PowerShell 실행 정책 오류

관리자 권한으로 PowerShell 실행 후:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
