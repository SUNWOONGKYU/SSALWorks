# SSAL Works Dev Package 설치 및 초기화 안내

> **대상:** 빌더 계정 사용자
> **범위:** 패키지 다운로드 → 도구 설치 → 프로젝트 초기화
> **버전:** 1.0

---

## 패키지만으로는 개발이 불가능합니다

SSAL Works Dev Package는 **폴더 구조와 템플릿**만 제공합니다.

**실제 개발을 시작하려면:**
1. 패키지 다운로드
2. 필수 도구 설치 (Git, Node.js, Claude Code 등)
3. 프로젝트 초기화 (git init, npm init, npm install)

위 3단계를 모두 완료해야 개발이 가능합니다.

---

## STEP 1: 패키지 다운로드

### 다운로드 경로

```
My Page > 자료 다운로드 > "SSAL Works Dev Package 다운로드" 카드 > "Dev Package 다운로드 (ZIP)" 버튼 클릭
```

```
SSAL_Works_Dev_Package/

├── assets/                 ← 정적 자원 (빈 폴더)

│   ├── commands/           ← 슬래시 명령어
│   └── work_logs/          ← 작업 기록 (빈 폴더)
│
├── Human_ClaudeCode_Bridge/
```

### 다운로드 후 할 일

1. ZIP 파일 압축 해제
2. 원하는 위치에 폴더 이동 (예: `C:\Projects\내프로젝트`)
3. 폴더 이름을 프로젝트에 맞게 변경

## STEP 2: 필수 도구 설치

### 설치 순서

```
1. Git 설치
     ↓
2. Node.js 설치 (npm 포함)
     ↓
3. VS Code 설치
     ↓
4. Claude Code 설치
     ↓
5. GitHub CLI 설치
     ↓
6. Vercel CLI 설치
     ↓
7. Supabase 설정
```

---

### 2-1. Git (필수)

**Git이 하는 일:**

- 모든 작업 내역을 자동 저장 (언제든 되돌리기 가능)
- 커밋과 동시에 자동 실행 (Pre-commit Hook):
  - Order Sheets MD → JS 파일로 번들링
  - Briefings MD → JS 파일로 번들링
  - Stage 폴더 → 배포 폴더로 자동 복사

**다운로드:**

| OS | 다운로드 링크 |
|----|--------------|
| **Windows** | https://git-scm.com/download/win |
| **Mac** | https://git-scm.com/download/mac |
| **Linux** | https://git-scm.com/download/linux |

**설치 확인:**
```bash
git --version
```

---

### 2-4. Claude Code

AI 개발 어시스턴트입니다.

**설치 명령어:**
```bash
npm install -g @anthropic-ai/claude-code
```

**설치 확인:**
```bash
claude --version
```

**API 키 설정:**

1. https://console.anthropic.com/ 접속
2. API Keys 메뉴에서 키 생성
3. 환경변수 설정:

```bash
# Windows (PowerShell)
$env:ANTHROPIC_API_KEY="your-api-key"

# Mac/Linux
export ANTHROPIC_API_KEY="your-api-key"
```

### 2-7. Supabase

**가입 및 프로젝트 생성:**

1. https://supabase.com 접속
2. 회원가입 (GitHub 계정으로 가능)
3. New Project 생성
4. 프로젝트 Settings > API에서 키 확인

**필요한 키:**

| 키 | 용도 |
|----|------|
| **Project URL** | 프로젝트 주소 |
| **anon public** | 클라이언트용 공개 키 |
| **service_role** | 서버용 비밀 키 (노출 금지) |

**환경변수 설정:**

프로젝트 루트에 `.env` 파일 생성:
```
SUPABASE_URL=your-project-url
SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
```

## STEP 3: 프로젝트 초기화

패키지 폴더로 이동한 후 순서대로 실행합니다.

### 3-1. Git 저장소 초기화

```bash
git init
```

### 3-2. npm 프로젝트 초기화

```bash
npm init -y
```

### 3-3. Claude Code 실행 및 초기화 요청

```bash
claude
```

Claude Code에게 요청:
```
"프로젝트 초기화해 줘. .gitignore 파일 만들고, 필요한 npm 패키지 설치해 줘."
```

### 초기화 후 생성되는 파일

| 파일 | 생성 방법 | 용도 |
|------|----------|------|
| `.git/` | `git init` | Git 버전 관리 |
| `package.json` | `npm init -y` | npm 프로젝트 설정 |
| `.gitignore` | Claude Code 요청 | Git 제외 파일 목록 |
| `node_modules/` | `npm install` | 설치된 패키지들 |

## 문제 해결

### "command not found" 오류

- 터미널을 껐다가 다시 열어보세요
- 설치 후 시스템 재시작이 필요할 수 있습니다

### npm 권한 오류 (Mac/Linux)

```bash
sudo npm install -g @anthropic-ai/claude-code
```

### Windows에서 PowerShell 실행 정책 오류

관리자 권한으로 PowerShell 실행 후:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
