# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 주요 사용자 플로우 설계
   - 회원가입 플로우
   - 로그인 플로우
   - 핵심 기능 플로우
   - 결제 플로우 (해당 시)

2. 플로우 설계 요소
   - 시작점/종료점
   - 단계별 액션
   - 분기 조건
   - 예외 처리

3. 플로우 다이어그램 작성
   - Mermaid 형식 또는 텍스트 다이어그램
   - Happy Path + Edge Cases

---

## A3. AI 작업 순서 (5단계)

### 1단계: Order Sheet 완전 이해

**체크리스트**:
- [ ] A2 작업 내용 확인
- [ ] A4 산출물 확인
- [ ] PART_B 특별 지시사항/참고사항 확인

**출력**: `'Order Sheet 확인 완료. 단계: P2-2-2'`

---

### 2단계: 문의사항 질문

**질문 형식**:
```
[P2-2-2] 질문: {내용}
옵션 A: {옵션1}
옵션 B: {옵션2}
```

**출력**: 질문 목록 또는 `'질문 없음'`

---

### 3단계: 실행 (Execution)

**체크리스트**:
- [ ] 회원가입/로그인 플로우
- [ ] 핵심 기능 플로우
- [ ] 예외 처리 정의
- [ ] 다이어그램 작성

---

### 4단계: 검증 (Verification)

**체크리스트**:
- [ ] 모든 주요 플로우가 설계되었는가?
- [ ] 예외 상황이 처리되었는가?
- [ ] 플로우 간 일관성이 있는가?
- [ ] 사용자 관점에서 자연스러운가?

**출력**: `'검증 완료'`

---
