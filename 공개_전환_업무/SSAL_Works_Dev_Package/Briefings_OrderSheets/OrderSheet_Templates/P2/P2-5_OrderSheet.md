# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 서비스 제공 가치 정의
   - SSAL Works가 제공하는 핵심 가치 정리
   - 고객이 얻는 혜택 명시
   - 차별화 포인트 강조

2. 타겟 고객 정의
   - 주요 타겟 고객군 식별
   - 고객 페르소나 작성
   - 고객의 Pain Point와 해결책

3. 서비스 소개 콘텐츠 작성
   - 랜딩 페이지용 카피
   - 마케팅용 핵심 메시지
   - FAQ 초안
