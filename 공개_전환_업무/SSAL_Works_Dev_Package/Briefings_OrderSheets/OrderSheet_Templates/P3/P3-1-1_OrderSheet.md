# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 프로토타입 구현
   - 지정된 화면 HTML/CSS/JS 구현
   - 디자인 가이드라인 적용
   - 반응형 레이아웃 구현

2. 공통 컴포넌트 작성
   - 헤더, 푸터, 사이드바
   - 버튼, 입력 필드 등 UI 컴포넌트
   - 모달, 토스트, 로딩 등 인터랙티브 요소

3. 기본 인터랙션 구현
   - 네비게이션 동작
   - 폼 인터랙션 (유효성 검사)
   - 상태 변화 시각화
