## 왜 개발 환경 설정이 필요한가요?

SSAL Works에서 웹사이트를 개발하려면 **컴퓨터에 개발 도구**가 설치되어 있어야 합니다.

| 구분 | 설명 |
|------|------|
| **개발 패키지** | 프로젝트 폴더 구조, 설정 파일, 템플릿 등 |
| **개발 도구** | Git, Node.js, npm 등 실행 환경 |
| **AI 도구** | Claude Code (AI 개발 어시스턴트) |

### 다운로드 후 할 일

1. ZIP 파일 압축 해제
2. 원하는 위치에 폴더 이동 (예: `C:\Projects\내프로젝트`)
3. 폴더 이름을 프로젝트에 맞게 변경

## STEP 2: 개발 도구 설치

패키지만으로는 개발이 안 됩니다. **개발 도구**도 함께 설치해야 합니다.

### 설치가 필요한 도구

| 도구 | 용도 | 설치 방법 |
|------|------|----------|
| **Git** | 버전 관리 | Claude Code에게 요청 |
| **Node.js** | JavaScript 실행 환경 | Claude Code에게 요청 |
| **npm 패키지** | 외부 라이브러리 | Claude Code에게 요청 |
| **Claude Code** | AI 개발 어시스턴트 | 별도 설치 필요 |

### Claude Code 설치

터미널(명령 프롬프트)에서 다음 명령어 실행:

```bash
npm install -g @anthropic-ai/claude-code
```

> **참고:** Node.js가 먼저 설치되어 있어야 합니다.

---

## STEP 3: Claude Code로 한 번에 설치

패키지 폴더에서 Claude Code를 실행하면, 나머지 도구를 자동으로 설치해줍니다.

### 실행 방법

1. **패키지 폴더로 이동**
   - 압축 해제한 폴더를 찾아서 열기

2. **터미널 열기**
   - 폴더 내에서 마우스 우클릭
   - "터미널에서 열기" 또는 "Git Bash Here" 선택

3. **Claude Code 실행**
   ```bash
   claude
   ```

4. **설치 요청**
   ```
   "프로젝트 개발 환경 설정을 위한 필수 도구 다 설치해 줘"
   ```

### Claude Code가 수행하는 작업

1. Git 설치 여부 확인 → 없으면 설치 안내
2. Node.js 설치 여부 확인 → 없으면 설치 안내
3. npm 패키지 설치 (`npm install`)
4. 프로젝트 초기화 완료 보고
