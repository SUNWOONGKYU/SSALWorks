매뉴얼 제공 시스템

확정일: 2025-12-12 버전: 2.0 방식: Markdown → GitHub Push → GitHub Actions 자동 변환 → HTML → Supabase DB

1. 시스템 개요

1.1 목적

  • 플랫폼 사용 매뉴얼을 체계적으로 제공
  • 기능별/주제별로 분류된 도움말 제공
  • 매뉴얼 작성 및 배포 자동화

1.2 학습용 콘텐츠와의 차이점

구분 학습용 콘텐츠 매뉴얼
목적 웹개발 지식 습득 플랫폼 사용법 안내
대상 개발자/학습자 플랫폼 사용자
내용 HTML, Supabase, Git 등 프로젝트 등록, 구독 관리 등
원본 위치 학습용_콘텐츠/ Project-SSAL-Grid/manual/
DB 테이블 learning_contents manuals

1.3 핵심 원칙

  • 자동화 우선: 학습용 콘텐츠와 동일하게 GitHub Actions 기반 처리
  • 단일 소스: MD 파일이 유일한 원본 (Single Source of Truth)
  • 분리 저장: 콘텐츠(GitHub) / 메타데이터(Supabase DB) 분리

5. 데이터베이스 스키마

5.1 manuals 테이블

CREATE TABLE manuals (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),

    -- 분류 체계
    category VARCHAR(100) NOT NULL,     -- 대분류
    subcategory VARCHAR(100),           -- 소분류

    -- 매뉴얼 정보
    title VARCHAR(255) NOT NULL,        -- 제목
    description TEXT,                   -- 설명
    url VARCHAR(500) NOT NULL,          -- GitHub raw URL (.html)

    -- 정렬 및 관리
    display_order INT DEFAULT 0,        -- 표시 순서
    is_active BOOLEAN DEFAULT true,     -- 활성화 여부

    -- 원본 파일 추적
    source_path VARCHAR(500),           -- MD 파일 경로
    content_hash VARCHAR(64),           -- MD 파일 해시 (변경 감지용)

    -- 타임스탬프
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 인덱스
CREATE INDEX idx_manuals_category ON manuals(category);
CREATE INDEX idx_manuals_display_order ON manuals(display_order);

5.2 URL 형식

jsdelivr CDN (권장):

https://cdn.jsdelivr.net/gh/{owner}/{repo}@{branch}/Project-SSAL-Grid/manual/파일명.html
https://cdn.jsdelivr.net/gh/{owner}/{repo}@{branch}/Project-SSAL-Grid/manual/파일명.md

예시 (SSALWorks):

https://cdn.jsdelivr.net/gh/SUNWOONGKYU/SSALWorks@master/Project-SSAL-Grid/manual/PROJECT_SSAL_GRID_MANUAL.html
https://cdn.jsdelivr.net/gh/SUNWOONGKYU/SSALWorks@master/Project-SSAL-Grid/manual/PROJECT_SSAL_GRID_MANUAL.md

주의: GitHub raw URL(raw.githubusercontent.com)은 Content-Type이 text/plain이라 iframe에서 HTML 렌더링이 안 됨. jsdelivr CDN 사용 권장.

9. 프로토타입 매뉴얼 뷰어

9.1 연결 구조

index.html (메인 페이지)
    └─ openManual() 함수
        └─ ../../../Project-SSAL-Grid/manual/manual.html (마크다운 뷰어)
            └─ jsdelivr CDN에서 PROJECT_SSAL_GRID_MANUAL.md fetch
                └─ Marked.js로 HTML 렌더링

9.2 로컬 CORS 문제 해결

문제: 브라우저에서 file:// 프로토콜로 열면 fetch 요청이 CORS 정책에 의해 차단됨.

해결: manual.html에서 로컬 파일 대신 jsdelivr CDN URL 사용

// 변경 전 (CORS 오류 발생)
const MANUAL_PATH = 'PROJECT_SSAL_GRID_MANUAL.md';

// 변경 후 (정상 작동)
const MANUAL_PATH = 'https://cdn.jsdelivr.net/gh/SUNWOONGKYU/SSALWorks@master/Project-SSAL-Grid/manual/PROJECT_SSAL_GRID_MANUAL.md';
