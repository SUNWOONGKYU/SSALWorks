# P1-3 비즈니스 모델 안내문

> **단계**: P1-3
> **버전**: 5.4

---

## 개요

수익 창출 방법과 비즈니스 구조를 정의하는 작업입니다.
비즈니스 모델 캔버스를 통해 사업의 전체 구조를 설계합니다.

---

## 목적

- 수익 모델 정의 및 가격 정책 수립
- 비용 구조 분석
- 손익 분기점(BEP) 분석
- 비즈니스 지속가능성 검증

---

## 주요 내용

### 비즈니스 모델 캔버스

| 항목 | 설명 |
|------|------|
| 고객 세그먼트 | 누구에게 가치를 제공하는가? |
| 가치 제안 | 어떤 문제를 해결하는가? |
| 채널 | 어떻게 고객에게 도달하는가? |
| 고객 관계 | 어떤 관계를 구축하는가? |
| 수익원 | 어떻게 수익을 창출하는가? |
| 핵심 자원 | 필요한 자원은 무엇인가? |
| 핵심 활동 | 핵심적인 활동은 무엇인가? |
| 핵심 파트너 | 필요한 파트너는 누구인가? |
| 비용 구조 | 주요 비용은 무엇인가? |

### 수익 모델

| 항목 | 예시 |
|------|------|
| 구독형 | 월/연 구독료 |
| 프리미엄 | 기본 무료 + 유료 기능 |
| 광고 | 광고 수익 |
| 수수료 | 거래 수수료 |

### 손익 분기점

| 항목 | 설명 |
|------|------|
| BEP | 손익이 0이 되는 지점 |
| 고정 비용 | 매출과 무관한 비용 |
| 변동 비용 | 매출에 비례하는 비용 |
