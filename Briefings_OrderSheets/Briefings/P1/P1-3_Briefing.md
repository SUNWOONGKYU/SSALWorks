# P1-3 비즈니스 모델

---

## 이 단계는 무엇인가요?

**수익 창출 방법**과 **비즈니스 구조**를 설계합니다.

좋은 아이디어도 돈을 벌 수 있어야 지속 가능합니다. 어떻게 수익을 낼 것인지 구체적으로 계획하는 단계입니다.

---

## 왜 필요한가요?

- **수익 모델 명확화**: 어떻게 돈을 벌 것인지 정합니다
- **비용 파악**: 운영에 얼마가 드는지 알 수 있습니다
- **손익 분기점**: 언제 흑자가 되는지 예측할 수 있습니다

---

## 무엇을 설계하나요?

### 비즈니스 모델 캔버스

| 영역 | 질문 |
|------|------|
| 고객 세그먼트 | 누구를 위한 것인가? |
| 가치 제안 | 어떤 가치를 주는가? |
| 수익원 | 어떻게 돈을 버는가? |
| 비용 구조 | 어디에 돈이 드는가? |

### 수익 모델

| 모델 | 예시 |
|------|------|
| 구독료 | 월/연 정기 결제 |
| 수수료 | 거래당 수수료 |
| 광고 | 광고 수익 |
| 프리미엄 | 기본 무료 + 유료 기능 |

### 손익 분기점 (BEP)

- 고정 비용 vs 변동 비용
- 목표 매출
- 흑자 전환 시점

---

|------|------|

| 가격대 | 예상 가격 범위 |
| 비용 항목 | 주요 비용 요소 |
