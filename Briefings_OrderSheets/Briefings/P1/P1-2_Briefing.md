# P1-2 시장 분석 안내문

> **단계**: P1-2
> **버전**: 5.4

---

## 개요

목표 시장과 경쟁 환경을 분석하는 작업입니다.
TAM/SAM/SOM 분석, 고객 페르소나, 경쟁 분석을 통해 시장 기회를 파악합니다.

---

## 목적

- 전체 시장 규모 파악 (TAM/SAM/SOM)
- 목표 고객 세그먼트 정의
- 경쟁 환경 분석 및 차별화 전략 수립
- 시장 트렌드 및 기회 요소 식별

---

## 주요 내용

### 시장 규모 분석

| 항목 | 설명 |
|------|------|
| TAM | 전체 시장 규모 (Total Addressable Market) |
| SAM | 접근 가능 시장 (Serviceable Available Market) |
| SOM | 실제 목표 시장 (Serviceable Obtainable Market) |

### 목표 고객

| 항목 | 설명 |
|------|------|
| 세그먼트 | 주요 타겟 고객 그룹 |
| 페르소나 | 3-5개의 구체적인 고객 프로필 |
| Pain Points | 고객이 겪는 문제점 |

### 경쟁 분석

| 항목 | 설명 |
|------|------|
| 직접 경쟁사 | 동일 시장의 경쟁자 |
| 간접 경쟁사 | 대체재를 제공하는 경쟁자 |
| SWOT | 강점/약점/기회/위협 분석 |
