# P1-1 Vision & Mission 안내문

> **단계**: P1-1
> **버전**: 5.4

---

## 개요

프로젝트의 비전과 미션을 정의하는 작업입니다.
사업의 방향성과 목표를 명확히 설정합니다.

---

## 목적

- 프로젝트의 궁극적인 목표 설정
- 팀의 공통된 방향성 확립
- 의사결정의 기준점 마련
- 이해관계자와의 커뮤니케이션 기반 구축

---

## 주요 내용

### Vision (비전)

| 항목 | 설명 |
|------|------|
| 정의 | 프로젝트가 궁극적으로 달성하고자 하는 미래상 |
| 형식 | 1-2문장으로 간결하게 |

### Mission (미션)

| 항목 | 설명 |
|------|------|
| 정의 | 비전을 달성하기 위한 구체적인 역할 |
| 포함 | 핵심 차별점, 타겟 사용자 |

### Core Values (핵심 가치)

- 프로젝트가 추구하는 원칙과 신념 (3~5개)

### Elevator Pitch

- 30초 내에 프로젝트를 설명할 수 있는 소개 문구
