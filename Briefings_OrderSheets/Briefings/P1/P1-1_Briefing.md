# P1-1 Vision & Mission 안내문

## 개요

프로젝트의 비전과 미션을 정의하는 작업입니다.
사업의 방향성과 목표를 명확히 설정합니다.

## 목적

- 프로젝트의 궁극적인 목표 설정
- 팀의 공통된 방향성 확립
- 의사결정의 기준점 마련
- 이해관계자와의 커뮤니케이션 기반 구축

## 주요 내용

### Vision (비전)
- 프로젝트가 궁극적으로 달성하고자 하는 미래상
- 장기적인 목표와 방향

### Mission (미션)
- 비전을 달성하기 위한 구체적인 역할
- 프로젝트가 제공하는 핵심 가치

### 핵심 가치
- 프로젝트가 추구하는 원칙과 신념

## 예상 산출물

| 산출물 | 설명 |
|--------|------|
| Vision Statement | 비전 선언문 |
| Mission Statement | 미션 선언문 |
| Core Values | 핵심 가치 목록 |

- **실행 유형**: Human-AI (PO 입력 필요)
- **의존성**: P0-1 완료
- **필요 정보**: 사업 아이디어, 목표 고객, 해결하려는 문제
