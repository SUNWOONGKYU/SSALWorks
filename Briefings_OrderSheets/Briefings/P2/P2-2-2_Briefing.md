# P2-2-2 사용자 플로우 안내문

> **단계**: P2-2-2
> **버전**: 5.4

---

## 개요

사용자 흐름(User Flow)을 설계하는 작업입니다.
사용자가 서비스를 이용하는 경로를 시각화합니다.

---

## 목적

- 사용자 경험 설계
- 화면 간 이동 경로 정의
- 예외 상황 처리 설계
- 개발 가이드 제공

---

## 주요 내용

### User Flow 유형

| 플로우 | 설명 |
|--------|------|
| 회원가입 | 신규 사용자 등록 |
| 로그인 | 기존 사용자 인증 |
| 핵심 기능 | 서비스 주요 기능 |
| 결제 | 유료 서비스 결제 |

### 플로우 요소

| 요소 | 설명 |
|------|------|
| 시작점 | 플로우 진입 지점 |
| 종료점 | 플로우 완료 지점 |
| 분기 | 조건에 따른 경로 분기 |
| 예외 | 오류 상황 처리 |
