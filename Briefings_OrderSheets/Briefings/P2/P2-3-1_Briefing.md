# P2-3-1 디자인 가이드라인 안내문

> **단계**: P2-3-1
> **버전**: 5.4

---

## 개요

UI/UX 디자인 가이드라인을 수립하는 작업입니다.
일관된 사용자 경험을 위한 디자인 원칙을 정의합니다.

---

## 목적

- 디자인 일관성 확보
- 브랜드 아이덴티티 반영
- 개발 효율성 향상
- 사용자 경험 품질 보장

---

## 주요 내용

### 컬러 시스템

| 항목 | 설명 |
|------|------|
| Primary | 주요 브랜드 컬러 |
| Secondary | 보조 컬러 |
| Semantic | 성공/경고/오류/정보 |

### 타이포그래피

| 항목 | 설명 |
|------|------|
| 폰트 | 웹폰트 또는 시스템 폰트 |
| 사이즈 | H1~H6, Body, Caption |
| 간격 | 라인 높이, 자간 |

### 컴포넌트

| 항목 | 예시 |
|------|------|
| 버튼 | Primary, Secondary, Outline |
| 입력 필드 | Default, Focus, Error |
| 카드/모달 | 공통 스타일 |
