# P2-3-3 프로토타입 제작

---

## 이 단계는 무엇인가요?

**실제로 동작하는 시제품**을 만듭니다.

클릭하면 화면이 바뀌고, 입력하면 반응하는 간단한 버전의 서비스를 만들어서 사용성을 검증하는 단계입니다.

---

## 왜 필요한가요?

- **사용성 검증**: 실제로 써보면서 문제점을 발견합니다
- **이해 공유**: 완성된 모습을 미리 체험할 수 있습니다
- **명세 확정**: 프로토타입을 기준으로 개발합니다

---

## 무엇을 만드나요?

### 프로토타입 범위

| 항목 | 설명 |
|------|------|
| 핵심 기능 | 주요 기능의 동작 |
| 화면 전환 | 페이지 간 이동 |
| 폼 입력 | 입력 후 반응 |

### 테스트 항목

| 항목 | 설명 |
|------|------|
| 사용성 | 쓰기 편한가 |
| 내비게이션 | 원하는 곳에 쉽게 가는가 |
| 피드백 | 사용자 의견 |

---

## PO가 결정할 것

| 항목 | 설명 |
|------|------|
| 프로토타입 범위 | 어디까지 동작하게 만들지 |
| 핵심 인터랙션 | 반드시 동작해야 하는 것 |
| 테스트 대상 | 누구에게 테스트할지 |
