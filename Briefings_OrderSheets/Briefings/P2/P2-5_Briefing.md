# P2-5 서비스 소개

---

## 이 단계는 무엇인가요?

**서비스의 핵심 가치**를 정리합니다.

우리 서비스가 고객에게 어떤 가치를 주는지, 왜 선택해야 하는지를 명확하게 정리하는 단계입니다.

---

## 왜 필요한가요?

- **가치 명확화**: 서비스의 핵심 가치를 한눈에 파악합니다
- **마케팅 기반**: 홍보 자료의 기초가 됩니다
- **방향 정립**: 개발 과정에서 우선순위를 정하는 기준이 됩니다

---

## 무엇을 정리하나요?

### 서비스 가치

| 항목 | 설명 |
|------|------|
| 핵심 가치 | 고객에게 주는 주요 혜택 |
| 차별점 | 경쟁 서비스와 다른 점 |
| 타겟 고객 | 누구를 위한 서비스인가 |

### 메시지 정리

| 항목 | 설명 |
|------|------|
| 한 줄 소개 | 서비스를 한 문장으로 |
| 주요 기능 | 3~5개 핵심 기능 |
| 사용 시나리오 | 어떻게 사용하는가 |

---

## PO가 결정할 것

| 항목 | 설명 |
|------|------|
| 핵심 가치 | 고객에게 줄 가장 중요한 가치 |
| 차별화 포인트 | 우리만의 강점 |
| 타겟 메시지 | 고객에게 전달할 핵심 메시지 |
