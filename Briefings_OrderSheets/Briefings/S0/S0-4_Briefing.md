# S0-4 SAL Grid Viewer 개발

---

## 이 단계는 무엇인가요?

**Grid 현황을 보여주는 대시보드**를 만듭니다.

웹에서 프로젝트 진행 상황을 한눈에 볼 수 있는 화면을 개발하는 단계입니다.

---

## 왜 필요한가요?

- **시각화**: 진행 상황을 그래프와 표로 봅니다
- **모니터링**: 실시간으로 현황을 파악합니다
- **보고**: 이해관계자에게 쉽게 설명할 수 있습니다

---

## 무엇을 만드나요?

### 뷰어 기능

| 항목 | 설명 |
|------|------|
| 매트릭스 보기 | Stage/Area별 현황 표 |
| 상태 표시 | 진행률 및 상태 색상 |
| 필터링 | Stage/Area별 필터 |
| 상세 보기 | Task 정보 팝업 |

### 대시보드

| 항목 | 설명 |
|------|------|
| 요약 통계 | 전체 진행률 |
| Stage별 현황 | 단계별 완료율 |
| Task 목록 | 개별 Task 상태 |
