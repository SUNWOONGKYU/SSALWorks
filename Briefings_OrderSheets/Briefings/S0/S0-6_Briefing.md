# S0-6 Verification Instructions 작성

---

## 이 단계는 무엇인가요?

**각 Task의 검증 기준**을 만듭니다.

Task가 제대로 완료되었는지 확인하는 체크리스트를 작성하는 단계입니다.

---

## 왜 필요한가요?

- **품질 보장**: 빠뜨린 것 없이 확인합니다
- **객관적 기준**: 누가 검증해도 같은 결과가 나옵니다
- **신뢰성**: 완료된 작업에 대한 신뢰가 생깁니다

---

## 무엇을 만드나요?

### Verification 내용

| 항목 | 설명 |
|------|------|
| 검증 항목 | 확인할 체크리스트 |
| 통과 기준 | 성공/실패 기준 |
| 실패 시 조치 | 문제 발견 시 대응 |

### 검증 원칙

| 원칙 | 설명 |
|------|------|
| 작성자 ≠ 검증자 | 다른 사람이 검증 |
| 명확한 기준 | 애매한 기준 금지 |
| 재현 가능 | 누가 해도 같은 결과 |

## PO가 결정할 것

| 항목 | 설명 |
|------|------|

| 우선 작성 Task | 먼저 검증 기준을 만들 Task |
