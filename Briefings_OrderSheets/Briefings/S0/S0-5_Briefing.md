# S0-5 Task Instructions 작성

---

## 이 단계는 무엇인가요?

**각 Task의 작업 지시서**를 만듭니다.

개별 Task를 어떻게 수행해야 하는지 상세하게 정리하는 단계입니다.

---

## 왜 필요한가요?

- **명확한 지시**: 무엇을 해야 하는지 분명해집니다
- **일관된 결과**: 누가 해도 같은 품질이 나옵니다
- **효율성**: 작업 시작 전 고민 시간이 줄어듭니다

---

## 무엇을 만드나요?

### Instruction 내용

| 항목 | 설명 |
|------|------|
| 작업 내용 | 수행할 작업 상세 |
| 작업 범위 | 포함/제외 범위 |
| 산출물 | 만들어야 할 것 |
| 참조 | 관련 문서 |

### 작성 기준

| 항목 | 설명 |
|------|------|
| 명확성 | 모호함 없이 |
| 구체성 | 실행 가능한 수준 |
| 완결성 | 빠진 내용 없이 |
