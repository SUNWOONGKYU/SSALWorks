# S0-5. Task Instructions 작성

---

## 개요

SAL Grid의 각 Task에 대한 상세 작업 지시서를 작성합니다.

---

## 이 단계에서 하는 일

| 항목 | 설명 |
|------|------|
| Task Instruction 작성 | 각 Task별 상세 작업 내용 정의 |
| 작업 범위 명시 | 수행할 작업의 범위와 한계 |
| 산출물 정의 | 생성해야 할 파일 목록 |

---

## Task Instruction 구조

```markdown
# Task ID: {Task ID}

## 작업 내용
- 수행할 작업 상세

## 산출물
- 생성할 파일 목록

## 참조
- 관련 문서/파일
```
