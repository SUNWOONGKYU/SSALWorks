# S0-3 SAL Grid Supabase 연동

---

## 이 단계는 무엇인가요?

**Grid 데이터를 데이터베이스에 저장**합니다.

SAL Grid 정보를 Supabase에 저장하여 언제든 조회할 수 있게 하는 단계입니다.

---

## 왜 필요한가요?

- **데이터 보존**: Grid 정보가 영구 저장됩니다
- **실시간 조회**: 웹에서 바로 현황을 볼 수 있습니다
- **팀 공유**: 여러 사람이 같은 정보를 봅니다

---

## 무엇을 만드나요?

### 데이터베이스

| 항목 | 설명 |
|------|------|
| Task 테이블 | Task 정보 저장 |
| Stage Gate 테이블 | 검증 상태 저장 |
| 보안 정책 | 접근 권한 설정 |

### 동기화

| 항목 | 설명 |
|------|------|
| 업로드 | 로컬 → DB |
| 다운로드 | DB → 로컬 |
| 자동 동기화 | 변경 시 자동 반영 |
