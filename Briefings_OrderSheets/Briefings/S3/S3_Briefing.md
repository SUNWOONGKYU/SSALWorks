# S3 개발 2차

## 단계 개요

**고급 기능을 개발**합니다.

AI 연동, 외부 서비스 통합 등 확장 기능을 구현하는 단계입니다.

## 필요성

- **기능 확장**: 핵심 기능 위에 고급 기능을 추가합니다
- **연동**: 외부 서비스와 연결합니다
- **경험 향상**: 사용자 경험을 더 좋게 만듭니다

## 작업 내용

### 작업 영역

- 백엔드: AI 인프라 및 고급 API
- 보안: 보안 강화
- 외부 연동: 외부 서비스 통합

## PO가 확인할 것

- 고급 기능: 확장 기능이 잘 동작하는가
- 연동: 외부 서비스 연결이 안정적인가
- 성능: 기능 추가 후에도 성능이 괜찮은가
