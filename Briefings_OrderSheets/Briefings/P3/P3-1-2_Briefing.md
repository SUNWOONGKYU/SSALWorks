# P3-1-2 페이지 추가

## 이 단계는 무엇인가요?

**개별 페이지**를 만듭니다.

기본 레이아웃 위에 각 기능별 페이지를 하나씩 추가하는 단계입니다.

## 왜 필요한가요?

- **전체 구현**: 모든 화면을 완성합니다
- **흐름 확인**: 페이지 간 이동이 자연스러운지 검증합니다
- **콘텐츠 배치**: 실제 내용이 들어갔을 때 모습을 확인합니다

## 무엇을 만드나요?

### 페이지 종류

- 홈/랜딩: 메인 페이지
- 인증: 로그인, 회원가입
- 대시보드: 사용자 메인 화면
- 설정: 계정 설정

### 구현 항목

- 레이아웃: 페이지별 구성
- 폼 요소: 입력 및 버튼
- 모달: 대화상자
- 상태 표시: 로딩, 오류 등

## PO가 결정할 것

- 추가할 페이지: 어떤 페이지를 만들지
- 우선순위: 먼저 만들 페이지
- 페이지별 기능: 각 페이지에서 할 수 있는 것
