# P3-3 유틸리티 스크립트

---

## 이 단계는 무엇인가요?

**자동화 스크립트**를 만듭니다.

반복 작업을 자동으로 처리하는 도구를 만들어서 개발과 운영을 편하게 하는 단계입니다.

---

## 왜 필요한가요?

- **자동화**: 반복 작업을 자동으로 처리합니다
- **일관성**: 매번 같은 방식으로 실행됩니다
- **효율성**: 시간과 노력을 절약합니다

---

## 무엇을 만드나요?

### 스크립트 종류

| 유형 | 설명 |
|------|------|
| 빌드 | 빌드 자동화 |
| 배포 | 배포 자동화 |
| 백업 | 데이터 백업/복원 |
| 테스트 | 테스트 자동화 |

### 도구

| 도구 | 용도 |
|------|------|
| Node.js | 스크립트 실행 |
| npm scripts | 패키지 명령어 |

---

## PO가 결정할 것

| 항목 | 설명 |
|------|------|
| 필요한 스크립트 | 어떤 자동화가 필요한가 |
| 입력/출력 | 스크립트가 무엇을 받고 내보내는가 |
| 실행 시점 | 언제 실행하는가 |
