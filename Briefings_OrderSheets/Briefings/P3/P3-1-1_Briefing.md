# P3-1-1 Frontend Prototype 안내문

## 개요

프론트엔드 프로토타입을 개발하는 작업입니다.
HTML/CSS/JavaScript로 실제 동작하는 화면을 구현합니다.

## 목적

- 실제 동작하는 UI 구현
- 디자인 검증
- 개발 기반 코드 생성
- 사용자 테스트 환경 구축

## 주요 내용

### 개발 범위
- 공통 레이아웃 (헤더, 푸터, 사이드바)
- 공통 컴포넌트 (버튼, 입력폼, 카드)
- 반응형 디자인

### 기술 스택
- HTML5
- CSS3 (또는 Tailwind CSS)
- Vanilla JavaScript

## 예상 산출물

| 산출물 | 설명 |
|--------|------|
| 공통 레이아웃 | header, footer, sidebar |
| 공통 CSS | 스타일 시트 |
| 기본 컴포넌트 | 재사용 가능한 UI 요소 |

## 실행 조건

- **실행 유형**: AI-Only
- **의존성**: P2-3 (디자인) 완료
