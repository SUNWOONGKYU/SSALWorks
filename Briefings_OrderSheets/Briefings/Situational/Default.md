# SSAL Works에 오신 것을 환영합니다

---

## SSAL Works란 무엇인가

SSAL Works는 **"Claude Code를 활용한 3차원 SAL Grid 기반 풀스택 웹사이트 개발 플랫폼"**입니다.

**Claude Code**는 Anthropic이 개발한 AI 도구입니다. 터미널에서 실행되며, 사용자의 지시를 받아 코드를 작성하고, 파일을 만들고, 프로젝트를 관리합니다. SSAL Works는 이 Claude Code를 핵심 도구로 사용합니다.

**풀스택 웹사이트**란 프론트엔드(사용자가 보는 화면)와 백엔드(서버, 데이터베이스, API)가 모두 갖춰진 완전한 웹사이트를 말합니다. 단순한 랜딩 페이지나 홍보용 웹사이트가 아니라, 회원가입, 로그인, 결제, 데이터 저장, 관리자 기능 등 실제 비즈니스를 운영할 수 있는 완전한 웹사이트입니다.

---

## SSAL이라는 이름의 의미

SSAL이라는 이름은 두 가지 의미가 결합되어 있습니다.

첫째, **S + SAL**입니다. S는 개발자 Sunny의 이름 첫 글자이고, SAL은 Stage, Area, Level의 약자로서 **3차원 좌표계**를 표현합니다.

둘째, **쌀**입니다. SSAL의 한글 발음은 "쌀"입니다. 쌀은 한국인에게 핵심 먹을거리입니다. SSAL Works는 마치 **황금쌀 방앗간**처럼, 창의적인 아이디어를 가져와서 돈을 버는 풀스택 웹사이트를 만드는 곳입니다.

빌더가 만드는 웹사이트는 사용자에게 돈을 벌어다주는 핵심 먹을거리가 됩니다.

## SAL: 3차원 프로젝트 관리 체계

SAL은 SSAL Works의 핵심 프로젝트 관리 체계입니다. 웹사이트 개발이라는 복잡한 작업을 **3차원 좌표계**로 구조화하여, 개발 지식이 없어도 전체 진행 상황을 한눈에 파악할 수 있게 합니다.

### Stage (X축) - 개발 단계

프로젝트는 시간 순서에 따라 5개 Stage를 거칩니다.

- **S1 (개발 준비)**: 개발 환경 설정, 데이터베이스 스키마 설계, 인증 시스템 기초 설정
- **S2 (개발 1차)**: 회원가입, 로그인, OAuth 연동 등 사용자 인증 기능 구현
- **S3 (개발 2차)**: AI 연동, 외부 서비스 연동 등 핵심 기능 구현
- **S4 (개발 3차)**: 결제 시스템, 관리자 기능, 크레딧 시스템 등 비즈니스 기능 구현
- **S5 (개발 마무리)**: QA 테스트, 버그 수정, 배포 및 안정화

### Area (Y축) - 작업 영역

각 Stage 안에서 작업은 11개 Area로 분류됩니다.

- **M (Documentation)**: 문서화 - API 명세서, 가이드 문서 작성
- **U (Design)**: UI/UX 디자인 - 화면 설계, 와이어프레임
- **F (Frontend)**: 프론트엔드 - 사용자가 보는 화면 개발
- **BI (Backend Infra)**: 백엔드 기반 - 서버 설정, 라이브러리 구성
- **BA (Backend APIs)**: 백엔드 API - 서버 로직, 데이터 처리
- **D (Database)**: 데이터베이스 - 테이블 설계, 쿼리 작성
- **S (Security)**: 보안 - 인증, 인가, 보안 설정
- **T (Testing)**: 테스트 - 단위 테스트, 통합 테스트
- **O (DevOps)**: 운영/배포 - CI/CD, 서버 운영
- **E (External)**: 외부 연동 - API 연동, 써드파티 서비스
- **C (Content)**: 콘텐츠 - 콘텐츠 관리, 데이터 입력

### Level (Z축) - 작업 순서

각 Task는 Stage별, Area별 순서를 나타냅니다.

### SAL Grid의 장점

이 3차원 체계를 통해 빌더는 다음을 할 수 있습니다.

- **현재 위치 파악**: "지금 S2 단계의 Frontend 영역을 작업 중이구나"
- **진행 상황 추적**: "전체 83개 Task 중 45개 완료, 54% 진행"
- **병목 구간 식별**: "S3의 External 연동에서 막혀있네"
- **의존성 관리**: "이 Task를 하려면 먼저 저 Task가 완료되어야 해"

개발 지식이 없어도 프로젝트 전체를 조감도처럼 내려다볼 수 있는 것, 그것이 SAL Grid의 핵심 가치입니다.

### 22개 Task 속성

각 Task는 22개의 속성으로 관리됩니다. Task ID, 이름, 목표부터 의존성, 생성 파일, 검증 결과, AI 의견까지. 이 체계적인 속성 관리를 통해 어떤 Task든 빠짐없이 추적하고, 문제가 생기면 즉시 원인을 파악할 수 있습니다.

## 작업 방식: 인간과 AI의 역할 분담

SSAL Works에서는 사용자와 AI가 역할을 분담합니다.

**AI(Claude Code)가 담당하는 것:** 코딩, 문법, 도구 사용, 반복 작업, 기술적 구현

**인간(사용자)이 담당하는 것:**
- 무엇을 만들 것인지 결정
- 방향 결정
- 비즈니스 모델 설계
- 최종 승인

쉽게 말해서, **"무엇을 만들 것인가"는 인간이 결정**하고, **"어떻게 만들 것인가"는 AI가 담당**합니다.

## 누구를 위한 플랫폼인가

SSAL Works는 다음과 같은 분들을 위한 플랫폼입니다.

- **스타트업 창업자**: 아이디어는 있지만 코딩을 할 줄 모르는 분
- **직장인 부업**: 퇴근 후와 주말을 활용해서 웹서비스를 만들고 싶은 분
- **은퇴 후 시니어**: 새로운 일거리를 찾고, 경험과 노하우를 살리고 싶은 분
- **기존 사업자**: 기존 사업에 맞춤형 웹사이트를 도입하고 싶은 분

---

## 빌더(Builder)란?

SSAL Works에서는 사용자를 **빌더(Builder)**라고 부릅니다.

빌더는 단순히 웹사이트만을 만드는 사람이 아니라, 웹사이트를 통해 비즈니스를 실현하는 사람입니다. 빌더는 기획하고, 의사결정하고, 방향을 정합니다. AI는 빌더의 지시에 따라 코딩을 수행합니다.

## 일반 회원으로 이용 가능한 기능

**무료 회원가입만으로** 다음 콘텐츠와 기능을 이용할 수 있습니다.

### 콘텐츠 열람 (무제한)

- **Books 80편**: 1권 Claude/Claude Code 사용법 (34편), 2권 풀스택 웹개발 기초지식 (34편), 3권 프로젝트 관리 방법 (12편)
- **Tips 65개**: 18개 카테고리별 실무 팁
- **외부 연동 가이드 5가지**: 데이터베이스 설정, 회원인증 설정, 이메일 시스템 설정, 배포/도메인 설정, 결제 시스템 설정

### 주요 기능

- **공지사항**: 플랫폼 공지사항 열람
- **My Page**: 프로필, 계정 정보 관리
- **인앱 알림**: 시스템 알림 수신
- **SAL Grid 예시 열람**: SSAL Works 프로젝트의 SAL Grid 구조 확인 (웹)
- **Order Sheet 템플릿 보기**: AI에게 작업 지시하는 32개 표준 양식 열람
- **Stage별 안내문 열람**: 각 개발 단계의 안내문 확인

→ **무료로 충분히 둘러본 후, 유용하다고 판단되면 빌더 계정을 개설하시면 됩니다.**

## 빌더 계정 개설자 추가 기능

빌더 계정을 개설하면 일반 회원 기능에 **추가로** 다음 기능을 이용할 수 있습니다.

### 프로젝트 관리

- **새로운 프로젝트 등록**: 본인의 웹사이트 프로젝트 생성
- **진행 프로세스 관리**: P0~S5 단계별 진행 상황 추적
- **Project SAL Grid 생성**: 본인 프로젝트용 SAL Grid 생성 및 관리

### 개발 도구

- **Order Sheet 전달하기**: 작성한 Order Sheet를 로컬에 저장하여 Claude Code에 전달
- **Reports 불러오기**: Claude Code 작업 결과 리포트 열람
- **Dev Package 다운로드**: AI 협업 개발 패키지 전체 다운로드
- **매뉴얼 다운로드**: PDF 파일로 다운로드 받아서 활용
- **예시 프로젝트 소스코드 및 문서**: SSAL Works 전체 소스코드 및 문서를 Google Drive를 통해 열람/다운로드

### AI Q&A 서비스

- **ChatGPT**: 코드 작성, 문서 생성, 일반 개발 질문
- **Gemini**: 코드 리뷰, 설계 관련 질문
- **Perplexity**: 검색 기반 정보, 최신 기술 동향

### 코칭 서비스

- **Sunny에게 질문하기**: 개발/코칭/자문 1:1 실시간 질의응답
- **3개월 1:1 맞춤형 코칭**: Sunny가 직접 코칭 (개발 + 창업 + 경영 자문)

## 3개월 1:1 맞춤형 코칭

빌더 계정을 개설하면, 처음 3개월 동안 Sunny가 직접 1:1로 맞춤형 코칭을 제공합니다.

### Sunny는 누구인가?

현직 공인회계사이자 스타트업 액셀러레이터입니다. 2025년 5월부터 7개월간 매일 평균 12시간 이상, 누적 2,500시간 이상을 AI와 함께 작업했습니다. 그 중 웹사이트 개발은 10월부터 시작해서, 3개월도 안 되는 기간에 풀스택 웹사이트 3개를 혼자서 완성했습니다.

### 코칭에서 다루는 내용

개발 코칭, 창업 코칭, 경영 자문

### 코칭의 가치

혼자 했으면 수개월 걸릴 일을 코칭을 통해 단축할 수 있습니다. 검색해도 안 나오는 실전 노하우, 삽질을 줄이는 방법, 올바른 방향 설정까지. 3개월 집중 코칭으로 빌더는 AI와 협업하는 개발 역량을 갖추게 됩니다.

## 빌더 계정 개설비

현재 개설비는 **300만원 (얼리버드, 부가세 포함)**입니다. 향후 **3,000만원까지** 단계적으로 인상될 예정입니다.

### 50% 환불 인센티브

얼리버드 300만원의 경우, 3개월 내에 웹사이트를 완성하고 서비스를 시작하여 고객 30명을 확보하면 개설비의 **50% (150만원)**를 환불해드립니다.

빌더의 성공이 곧 SSAL Works의 성공입니다. 열심히 하시는 분께 인센티브로 보답합니다.
