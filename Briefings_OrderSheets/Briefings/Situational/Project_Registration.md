# 새 프로젝트 등록

## 개요

SSAL Works에서 **새로운 웹사이트 프로젝트**를 시작합니다.

프로젝트를 등록하면 AI와 함께 단계별로 웹사이트를 개발할 수 있습니다.

## 등록 전 확인사항

프로젝트 등록을 위해 다음 조건이 필요합니다:

- **빌더 계정 개설**: 빌더 계정이 개설되어 있어야 합니다
- **진행 중인 프로젝트 없음**: 한 번에 하나의 프로젝트만 진행 가능합니다

## 입력 정보

### 프로젝트 이름 (필수)
만들고자 하는 웹사이트의 이름을 입력합니다.

### 프로젝트 설명 (선택)
어떤 웹사이트를 만들고 싶은지 간략히 설명합니다.

### 프로젝트 유형
- **포트폴리오**: 개인 작품, 이력 소개
- **블로그**: 콘텐츠 발행, 정보 공유
- **비즈니스**: 회사/서비스 소개
- **기타**: 자유 형식

## 등록 후 진행 사항

### 1. 개발 패키지 다운로드

프로젝트 개발에 필요한 폴더와 파일이 포함된 패키지입니다.

| 구분 | 설명 |
|------|------|
| 개발 패키지 | 프로젝트 폴더 구조, 설정 파일, 템플릿 등 |
| 개발 도구 | Git, Node.js, npm 등 실행 환경 |
| AI 도구 | Claude Code (AI 개발 어시스턴트) |

### 2. 개발 도구 설치

| 도구 | 용도 |
|------|------|
| Git | 버전 관리 |
| Node.js | JavaScript 실행 환경 |
| Claude Code | AI 개발 어시스턴트 |

### 3. Claude Code로 개발 시작

패키지 폴더에서 Claude Code를 실행하면 개발을 시작할 수 있습니다.

## 5단계 개발 프로세스

| 단계 | 이름 | 내용 |
|------|------|------|
| P0 | 작업 환경 | 디렉토리 구조 생성, 상태 관리 |
| P1 | 사업계획 | 시장조사, 경쟁분석, 사업계획서 |
| P2 | 프로젝트 기획 | 요구사항, 아키텍처, UI/UX 설계 |
| P3 | 프로토타입 | 페이지 개발, DB 구축 |
| S1-S5 | 본개발 | 인증, API, 결제, 배포 |

## 주의사항

- 프로젝트는 한 번에 하나만 진행할 수 있습니다
- 기존 프로젝트 완료/중단 후 새 프로젝트 시작 가능
