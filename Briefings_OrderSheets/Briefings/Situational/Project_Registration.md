# 개발 패키지 설치 안내

## 축하합니다!

프로젝트가 성공적으로 등록되었습니다.

이제 개발을 시작하기 위한 개발 환경 설정이 필요합니다.

---

## 왜 개발 환경 설정이 필요한가요?

SSAL Works에서 웹사이트를 개발하려면 컴퓨터에 개발 도구가 설치되어 있어야 합니다.

- **개발 패키지**: 프로젝트 폴더 구조, 설정 파일, 템플릿 등
- **개발 도구**: Git, Node.js, npm 등 실행 환경
- **AI 도구**: Claude Code (AI 개발 어시스턴트)

---

## STEP 1: 개발 패키지 다운로드

프로젝트 개발에 필요한 폴더와 파일이 포함된 패키지입니다.

패키지에는 다음이 포함되어 있습니다:
- 표준 프로젝트 폴더 구조
- AI 작업 규칙 파일
- 자동화 스크립트
- 작업 지시서 템플릿

**다운로드 후 할 일:**
1. ZIP 파일 압축 해제
2. 원하는 위치에 폴더 이동 (예: C:\Projects\내프로젝트)
3. 폴더 이름을 프로젝트에 맞게 변경

---

## STEP 2: 개발 도구 설치

패키지만으로는 개발이 안 됩니다. 개발 도구도 함께 설치해야 합니다.

**설치가 필요한 도구:**
- Git: 버전 관리
- Node.js: JavaScript 실행 환경
- npm 패키지: 외부 라이브러리
- Claude Code: AI 개발 어시스턴트

**Claude Code 설치 방법:**

터미널(명령 프롬프트)에서 다음 명령어 실행:

```bash
npm install -g @anthropic-ai/claude-code
```

(Node.js가 먼저 설치되어 있어야 합니다)

---

## STEP 3: Claude Code로 개발 시작

패키지 폴더에서 Claude Code를 실행하면, 나머지 도구를 자동으로 설치해줍니다.

**실행 방법:**

1. 패키지 폴더로 이동 (압축 해제한 폴더)
2. 폴더 내에서 터미널 열기 (마우스 우클릭 → "터미널에서 열기")
3. Claude Code 실행: `claude`
4. 설치 요청: "프로젝트 개발 환경 설정을 위한 필수 도구 다 설치해 줘"

**Claude Code가 수행하는 작업:**
- Git 설치 여부 확인 및 안내
- Node.js 설치 여부 확인 및 안내
- npm 패키지 설치
- 프로젝트 초기화 완료

---

## STEP 4: 설치 완료 후

모든 설치가 완료되면 Dashboard로 돌아와서 개발을 시작하세요.

**다음 단계:**

2. 안내문 읽기 - 각 단계별 Briefing 확인
3. Order Sheet 작성 - Claude Code에게 작업 지시
