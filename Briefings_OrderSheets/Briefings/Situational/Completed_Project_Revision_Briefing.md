# 완료된 프로젝트 수정

## 이 기능은 무엇인가요?

완료된 프로젝트를 다시 열어 **수정 작업**을 진행합니다.

버그 수정, 기능 개선, UI 변경 등 완료된 프로젝트에 추가 작업이 필요할 때 사용합니다.

## 어떤 경우에 사용하나요?

- 버그 수정: 발견된 오류나 문제 해결
- 기능 추가: 새로운 기능 추가
- 기능 개선: 기존 기능 성능/UX 개선
- UI 수정: 디자인, 레이아웃 변경
- 콘텐츠 수정: 텍스트, 이미지 등 수정

## 수정 작업 흐름

1. 수정할 내용 명확히 정리
2. Order Sheet에 수정 요청 작성
3. AI에게 Order Sheet 전달
4. 수정 결과 확인 및 테스트
5. 수정 완료 후 재배포 (필요시)

## 주의사항

- 백업 권장: 수정 전 현재 상태 백업 권장
- 범위 명확히: 수정 범위를 구체적으로 명시
- 테스트 필수: 수정 후 반드시 동작 테스트

> 확인 버튼을 클릭하면 수정 요청 Order Sheet가 Control Desk에 로드됩니다.

> **위의 작업을 위하여 준비된 Order Sheet 템플릿을 Control Desk에 로딩하시겠습니까?**
