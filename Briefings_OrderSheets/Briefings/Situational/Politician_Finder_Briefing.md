# PoliticianFinder 프로젝트

## 개요

PoliticianFinder는 대한민국 정치인 정보를 검색하고 분석할 수 있는 웹 서비스입니다.

SSAL Works 개발자 Sunny가 직접 개발한 완료된 프로젝트로, 예시 프로젝트로 제공됩니다.

## 주요 기능

- **정치인 검색**: 이름, 정당, 지역구별 검색
- **상세 정보**: 경력, 의정활동, 공약 등 상세 정보 제공
- **비교 분석**: 정치인 간 비교 기능
- **모바일 최적화**: 반응형 디자인 적용

## 기술 스택

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Vercel Serverless Functions
- **Database**: Supabase PostgreSQL
- **배포**: Vercel

## 활용 방법

1. 구조 파악: 프로젝트의 폴더 구조 및 아키텍처 파악
2. 문서 참고: 기획 문서, API 명세서 등 참고
3. 코드 활용: 실제 구현된 코드를 참고하여 유사 기능 구현
