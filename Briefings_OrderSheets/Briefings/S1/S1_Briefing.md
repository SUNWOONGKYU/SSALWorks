# S1 개발 준비

---

## 이 단계는 무엇인가요?

**본격적인 개발을 위한 준비**를 합니다.

개발 환경, 인프라, 보안, 테스트 기반을 구축하는 단계입니다.

---

## 왜 필요한가요?

- **환경 구축**: 개발에 필요한 도구와 환경을 준비합니다
- **기반 마련**: 안정적인 개발을 위한 토대를 만듭니다
- **표준 수립**: 개발 방식과 규칙을 정합니다

---

## 무엇을 준비하나요?

### 작업 영역

| 영역 | 내용 |
|------|------|
| 문서화 | 기술 문서 작성 |
| 프론트엔드 | 프론트엔드 기반 |
| 백엔드 | 백엔드 인프라 및 API 기초 |
| 데이터베이스 | DB 확장 및 최적화 |
| 보안 | 보안 정책 수립 |
| 테스트 | 테스트 환경 구축 |
| 배포 | 배포 파이프라인 |

---

## PO가 확인할 것

| 항목 | 설명 |
|------|------|
| 환경 설정 | 개발 환경이 제대로 구축되었는가 |
| 보안 정책 | 보안 기준이 적절한가 |
| 테스트 환경 | 테스트할 수 있는 환경인가 |
