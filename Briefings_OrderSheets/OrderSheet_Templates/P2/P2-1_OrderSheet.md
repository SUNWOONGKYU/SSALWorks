> **버전**: 5.4
> **단계**: P2-1 (프로젝트 개요 작성)
> **목적**: 프로젝트 전체 개요 및 범위 정의

---

# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 프로젝트 개요 작성
   - 프로젝트명 및 설명
   - 프로젝트 목표
   - 핵심 기능 요약

2. 프로젝트 범위 정의
   - 포함 범위 (In Scope)
   - 제외 범위 (Out of Scope)
   - 제약 사항 및 가정

3. 이해관계자 정의
   - 주요 이해관계자
   - 역할 및 책임

4. 마일스톤 정의
   - 주요 마일스톤
   - 완료 조건

---

## A3. AI 작업 순서 (5단계)

### 1단계: Order Sheet 완전 이해

**체크리스트**:
- [ ] A2 작업 내용 확인
- [ ] A4 산출물 확인
- [ ] PART_B 특별 지시사항/참고사항 확인

**출력**: `'Order Sheet 확인 완료. 단계: P2-1'`

---

### 2단계: 문의사항 질문

**질문 형식**:
```
[P2-1] 질문: {내용}
옵션 A: {옵션1}
옵션 B: {옵션2}
```

**출력**: 질문 목록 또는 `'질문 없음'`

---

### 3단계: 실행 (Execution)

**체크리스트**:
- [ ] 프로젝트 개요 작성
- [ ] 범위 정의
- [ ] 이해관계자 정의
- [ ] 마일스톤 정의

---

### 4단계: 검증 (Verification)

**체크리스트**:
- [ ] 프로젝트 개요가 명확하고 간결한가?
- [ ] 범위가 구체적으로 정의되었는가?
- [ ] 이해관계자가 빠짐없이 식별되었는가?
- [ ] 마일스톤이 측정 가능한가?

**출력**: `'검증 완료'`

---

### 5단계: 완료 보고 (Report)

**보고서 생성**:
- 파일명: `P2-1_completion_report.md`
- 저장 위치: `Human_ClaudeCode_Bridge/Reports/`

**보고 내용**:
- 완료된 작업 요약
- 생성된 문서
- 다음 단계 안내 (P2-2-1)
