# PART A: 표준 내용

## A1. AI 준수 사항

**AI가 반드시 지켜야 할 사항:**

1. 이 Order Sheet를 100% 이해할 때까지 작업 시작 금지
2. 규칙 파일(`.claude/rules/`) 확인 전 파일 생성/저장 금지
3. 불명확한 점은 추측 금지, 반드시 질문
4. 작업 순서 (A3 참조) 건너뛰거나 변경 금지
5. 거짓 기록 절대 금지

---

## A2. 작업 내용

**수행할 작업:**

1. 프로토타입 범위 정의
   - 구현할 인터랙션 목록
   - 연결할 화면 플로우
   - 시뮬레이션할 기능

2. 인터랙션 구현
   - 버튼 클릭, 폼 제출 등 기본 인터랙션
   - 화면 전환 애니메이션
   - 상태 변화 시뮬레이션 (로딩, 성공, 오류)

3. 테스트 시나리오 작성
   - 주요 사용자 여정 테스트
   - 예외 상황 테스트
   - 사용성 검증 포인트
