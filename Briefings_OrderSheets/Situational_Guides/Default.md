# SSAL Works 대시보드

---

## 개요

안녕하세요! 오늘도 SSAL Works와 함께 프로젝트를 진행해보세요.

---

## 빠른 시작

| 항목 | 설명 |
|------|------|
| 프로젝트 선택 | 좌측 사이드바에서 작업할 프로젝트 선택 |
| 진행 현황 | 현재 단계와 다음 작업 확인 |
| Order Sheet | AI에게 작업 요청 |

---

## 유용한 기능

- **예시 프로젝트** - 실제 완료된 프로젝트 참고
- **학습 콘텐츠** - 웹개발 기초 지식 학습
- **AI Q&A** - 궁금한 점 AI에게 질문

---

> **Tip:** 작업을 시작하려면 좌측에서 프로젝트를 선택하세요.
