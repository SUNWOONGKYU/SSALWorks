# 프로젝트 작업

---

## 개요

현재 프로젝트의 작업을 진행합니다.

---

## 작업 흐름

| 단계 | 설명 |
|------|------|
| 안내문 확인 | 현재 단계의 설명을 읽습니다 |
| Order Sheet 로드 | AI에게 전달할 작업 지시서 |
| 결과 확인 | AI의 작업 결과를 검토합니다 |

---

## 진행 순서

1. 좌측에서 현재 단계 선택
2. 안내문(Briefing) 읽기
3. 확인 버튼 클릭 → Order Sheet 로드
4. Order Sheet를 AI에게 전달
5. 결과물 확인 및 다음 단계 진행

---

> Order Sheet를 통해 AI와 체계적으로 협업하세요.
