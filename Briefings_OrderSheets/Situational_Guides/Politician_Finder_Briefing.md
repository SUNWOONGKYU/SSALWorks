# PoliticianFinder

---

## 개요

**AI 기반 정치인 평가 플랫폼**

PoliticianFinder는 AI 기술을 활용하여 정치인의 활동, 공약, 투표 기록 등을 분석하고 시민들에게 객관적인 정보를 제공하는 플랫폼입니다.

## 주요 기능

1. **정치인 검색** - 이름, 지역, 정당별 정치인 검색
2. **AI 분석** - 공약 이행률, 의정활동 분석
3. **비교 기능** - 정치인 간 비교 분석
4. **시민 평가** - 사용자 참여 평가 시스템

---

## 기술 스택

- Frontend: React, TypeScript
- Backend: Node.js, Express
- Database: PostgreSQL
- AI: OpenAI API
- Deployment: Vercel
