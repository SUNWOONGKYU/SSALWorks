# SSAL Works 예시 프로젝트 안내

---

## 프로젝트 소개

**SSAL Works**는 Claude Code를 활용한 **3차원 SAL Grid 기반 풀스택 웹사이트 개발 플랫폼**입니다.

AI와 사람이 협업하여 체계적으로 웹 프로젝트를 완성해 나가는 과정을 직접 확인하실 수 있습니다.

---

## 프로젝트 구조

| Stage | 폴더명 | 설명 |
|-------|--------|------|
| P0 | 작업_디렉토리_구조_생성 | 프로젝트 폴더 구조 설계 |
| P1 | 아이디어_수립 | 서비스 컨셉 및 아이디어 정리 |
| P2 | 프로젝트_기획 | 요구사항, UI/UX, 워크플로우 설계 |
| P3 | 프로토타입_제작 | 프론트엔드 프로토타입 개발 |
| S0 | Project-SAL-Grid_생성 | SAL Grid Task 설계 및 관리 |
| S1~S5 | 개발 단계 | 준비 → 1차 → 2차 → 3차 → 마무리 |

---

## 열람 가능 자료

### 1. 기획 문서
- 서비스 아이디어 및 컨셉
- 기능 요구사항 정의서
- UI/UX 목업 및 와이어프레임
- 사용자 플로우 다이어그램

### 2. SAL Grid
- 60개 Task의 체계적 분류
- Task Instruction / Verification 문서
- Stage Gate 검증 리포트

### 3. 소스 코드
- 프론트엔드 (HTML, CSS, JavaScript)
- 백엔드 API (Node.js, Vercel Functions)
- 데이터베이스 스키마 (Supabase)

### 4. Order Sheet 예시
- 각 단계별 실제 작성 예시
- AI에게 전달한 지시사항 확인
