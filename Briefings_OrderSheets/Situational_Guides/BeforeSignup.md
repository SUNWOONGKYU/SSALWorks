# SSAL Works 소개

---

## 개요

**SSAL Works**는 비개발자도 AI(Claude Code)와 협업하여 전문적인 웹사이트를 개발할 수 있는 플랫폼입니다.

---

## 주요 특징

| 특징 | 설명 |
|------|------|
| AI 기반 개발 | Claude Code가 코딩을 담당합니다 |
| 체계적인 프로세스 | 기획부터 배포까지 단계별 진행 |
| 실시간 협업 | Order Sheet로 AI에게 작업 요청 |
| 학습 콘텐츠 제공 | 웹개발 기초 지식 습득 가능 |
